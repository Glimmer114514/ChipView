## ځانګړتیاوې

- **ریښتیني وخت څارنه**: د CPU فریکونسي، د CPU تودوخه، د GPU فریکونسي، د GPU تودوخه، د VRAM کارونه، د حافظې کارونه
- **دوه ښودنې حالتونه**: د کونج ویجیټ / پورتنۍ پټه، په یوه کلیک بدلون
- **د ښودنې دودیز کول**: کوم معیارونه چې وښیاست خپل خوښه ټاکنه، د سلنې/اصلي ارزښتونو بدلون
- **تل پورتنۍ**: د ټولو کړکیو پورتنۍ برخه کې پاتې کیږي، په ټولې پرده لوبو کې کار کوي
- **خپلکار سایز**: د کړکۍ پلنوالی د منځپانګې سره سم خپلکاری
- **د سمون وړ ستایل**: د کړکۍ شفافیت، د شالید شفافیت، د فونټ اندازه ټول د سمون وړ
- **څو ژبه ملاتړ**: ۲۰+ ژبې، د سیسټم ژبه اتومات تشخیصوي
- **آسانه چلونه**: د کیڼ کش سره حرکت، د ښي کلیک مینو لپاره چټک تنظیمات، د سکرین کنجونو ته اتومات نښلېدل
- **د نمونې تنظیم**: ۲۰۰ms - ۳۰۰۰ms د سمون وړ
- **اتومات پیل**: د Windows ننوتلو پر مهال اتومات پیل
- **کم سرچینه کارونه**: په شالید کې لږ سرچینه کاروي

## چټک پیل

### مستقیم کارونه

`ChipPeek.exe` ښکته کړئ او دوه ځله کلیک وکړئ ترڅو پرمخ بوځي (د CPU تودوخې او دقیق فریکونسي لوستلو لپاره به اتومات د مدیر حقونه وغواړي).

### له سرچینې څخه پرمخ بیول

```bash
# ذخیره کلون کړئ
git clone https://github.com/Glimmer114514/ChipPeek.git
cd ChipPeek

# پاتې والونه نصب کړئ
pip install -r requirements.txt

# پرمخ بوځئ
python main.py
```

## د سیسټم اړتیاوې

- Windows 10 / Windows 11
- د مدیر حقونه (ورسپارل شوي)، که نه وی نو د CPU تودوخه او دقیق فریکونسي ممکن نشته
- .NET Framework 4.7.2 یا لوړ (د LibreHardwareMonitor لپاره اړین)

## د EXE په توګه جوړول

```bash
pip install pyinstaller
pyinstaller --noconfirm --onefile --windowed --name "ChipPeek" --icon "app.ico" --manifest "admin.manifest" --add-data "libs;libs" --add-data "app.ico;." --hidden-import "clr" --hidden-import "pynvml" --hidden-import "win32gui" --hidden-import "win32con" main.py
```

د جوړولو وروسته، د EXE فایل په `dist/ChipPeek.exe` کې دی.

## کارونه

### بنسټیزې کړنې

| کړنه | تشریح |
|------|------|
| کیڼ کش | د کړکۍ موقعیت حرکت |
| ښي کلیک | مینو پرانیستی (حالت بدلول / تنظیمات / وتل) |
| اتومات نښلېدل | د سکرین کنجونو څخه د ۲۰px په حدودو کې اتومات نښلېدل |

### ښودنې حالتونه

- **د کونج ویجیټ**: ټول معیارونه په عمودي توګه ترتیب شوي، کیدای شي په هر سکرین کونج کې ځای په ځای شي
- **پورتنۍ پټه**: ټول معیارونه په افقي توګه ترتیب شوي، په ډیفالټ سره د سکرین پورتنۍ برخه کې منځ کې

### تنظیمات

- **ښودنې حالت**: د کونج ویجیټ / پورتنۍ پټه
- **د ویجیټ موقعیت**: ښکته ښي / ښکته کیڼ / پورته ښي / پورته کیڼ
- **د کړکۍ شفافیت**: ۳۰٪ - ۱۰۰٪ ټولګړی د کړکۍ شفافیت
- **د شالید شفافیت**: ۰٪ - ۱۰۰٪ د شالید شفاف (لیکنه روښانه پاتې کیږي)
- **د نمونې فاصله**: ۲۰۰ms - ۳۰۰۰ms د ډاټا تازه کولو کچه
- **د فونټ اندازه**: ۸ - ۲۰ نقطه فونټ
- **ژبه**: ۲۰+ ژبې، د سیسټم ژبه اتومات تشخیصوي
- **ښودنې معیارونه**: هر معیار په جلا توګه بدلول
- **ښودنې بڼه**: VRAM / حافظه کولی شي د سلنې یا اصلي ارزښتونو تر منځ بدل کړي
- **اتومات پیل**: د Windows ننوتلو پر مهال اتومات پرمخ تګ

## ټیکنالوژۍ کړکۍ

- **GUI**：Tkinter
- **د سختوالي ډاټا**：LibreHardwareMonitorLib (له pythonnet سره)، psutil، pynvml (د NVIDIA GPU لپاره بدل)
- **د سیسټم مدغم کول**：pywin32 (د کړکۍ پورتنۍ، د اتومات پیل رجسټري)
- **بسته بندي**：PyInstaller

ChipPeek/

├── app.png                  # د آیکون مخکتنه
├── requirements.txt         # د Python پاتې والونه
├── docs/                    # څو ژبې اسناد
│   └── README_*.md

## د تنظیماتو فایل

`config.json` فایل په EXE سره د یوه لارښود کې خوندي کیږي، ټول د سمون وړ تنظیمات لري. تنظیمات په اتومات ډول هغه وخت خوندي کیږي کله چې بدل شي.
