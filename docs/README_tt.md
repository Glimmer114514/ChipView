## Мөмкинлекләр

- **Реаль вакытта күзәтү**: CPU ешлыгы, CPU температурасы, GPU ешлыгы, GPU температурасы, VRAM куллануы, хәтер куллануы
- **Ике күрсәтү рәвеше**: Почмак виджеты / Өске такта, бер басу белән алыштыру
- **Үзгәртелә торган күрсәтү**: күрсәтелә торган метрикларны ирекле сайлау, процент/чын бәя алыштыру
- **Һәрвакыт өстә**: барлык тәрәзәләр өстендә тора, тулы экранлы уеннарда эшли
- **Авто-зурлык**: тәрәзә киңлеге эчтәлеккә карап автоматик рәвештә үзгәрә
- **Үзгәртелә торган стиль**: тәрәзә тоныклыгы, фон үтә күренмәлелеге, хәреф зурлыгы үзгәртелә ала
- **Күптелле ярдәм**: 20+ тел, система телен автоматик рәвештә ачыклау
- **Уңайлы куллану**: сул сөйрәлү белән күчерү, уң басу менюсы белән тиз көйләүләр, экран читләренә автоматик ябышу
- **Конфигурацияләнә торган өлчәү**: 200ms - 3000ms үзгәртелә ала
- **Автоматик башлану**: Windows'ка кергәндә автоматик рәвештә башлана
- **Аз ресурслы куллану**: арткы планда минималь эз

## Тиз башлану

### Туры куллану

`ChipPeek.exe` файлын йөкләгез һәм ике тапкыр басып эшли башлагыз (CPU температурасы һәм теүәл ешлык укып алу өчен автоматик рәвештә администратор хокукларын сораячак).

### Чыганактан эшләтеп җибәрү

```bash
# Репозиторийны клонлау
git clone https://github.com/Glimmer114514/ChipPeek.git
cd ChipPeek

# Бәйләнешләрне урнаштыру
pip install -r requirements.txt

# Эшләтеп җибәрү
python main.py
```

## Система таләпләре

- Windows 10 / Windows 11
- Администратор хокуклары (тәкъдим ителә), анысыз CPU температурасы һәм теүәл ешлык булмаска мөмкин
- .NET Framework 4.7.2 яки югарырак (LibreHardwareMonitor тарафыннан таләп ителә)

## EXE итеп төзү

```bash
pip install pyinstaller
pyinstaller --noconfirm --onefile --windowed --name "ChipPeek" --icon "app.ico" --manifest "admin.manifest" --add-data "libs;libs" --add-data "app.ico;." --hidden-import "clr" --hidden-import "pynvml" --hidden-import "win32gui" --hidden-import "win32con" main.py
```

Төзелгәннән соң, EXE файлы `dist/ChipPeek.exe` адресында урнаша.

## Куллану

### Төп гамәлләр

| Гамәл | Тасвирлама |
|------|------|
| Сул сөйрәлү | Тәрәзәне күчерү |
| Уң басу | Меню ачу (рәвеш алыштыру / көйләүләр / чыгу) |
| Автоматик ябышу | Экран читләренә 20px эчендә автоматик ябешә |

### Күрсәтү рәвешләре

- **Почмак виджеты**: барлык метриклар вертикаль рәвештә урнаштырылган, экранның теләсә нинди почмагына куеп булала
- **Өске такта**: барлык метриклар горизонталь рәвештә урнаштырылган, әйтик экран өстендә үзәкләштерелгән

### Көйләүләр

- **Күрсәтү рәвеше**: Почмак виджеты / Өске такта
- **Виджет урыны**: Аста уңда / Аста сулда / Өстә уңда / Өстә сулда
- **Тәрәзә тоныклыгы**: гомуми тәрәзә үтә күренмәлелеге 30% - 100%
- **Фон үтә күренмәлелеге**: фон үтә күренмәлелеге 0% - 100% (текст ачык кала)
- **Өлчәү аралыгы**: мәгълүмат яңарту ешлыгы 200ms - 3000ms
- **Хәреф зурлыгы**: 8 - 20 пункт хәреф
- **Тел**: 20+ тел, система телен автоматик рәвештә ачыклау
- **Күрсәтелә торган метриклар**: һәр метриканы аерым үзгәртү
- **Күрсәтү форматы**: VRAM / Хәтер процент белән я чын бәя арасында алыштырып була
- **Автоматик башлану**: Windows'ка кергәндә автоматик рәвештә эшли башлау

## Технологик стек

- **График интерфейс**：Tkinter
- **Җиһаз мәгълүматы**：LibreHardwareMonitorLib (pythonnet аша), psutil, pynvml (NVIDIA GPU резерв чишелеше)
- **Система интеграциясе**：pywin32 (тәрәзә өстә, автостат регистры)
- **Урнаштыру**：PyInstaller

## Проект структурасы

```
ChipPeek/
├── main.py                  # Кереш ноктасы
├── monitor_window.py        # Тәрәзә UI һәм интерактив логика
├── hardware_monitor.py      # Җиһаз мәгълүматын җыю
├── config.py                # Конфигурация белән идарә итү
├── i18n.py                  # Халыкаралаштыру
├── generate_icon.py         # Иконка генерацияләү скрипты
├── admin.manifest           # UAC администратор хокукы манифесты
├── app.ico                  # Кушымта иконкасы
├── app.png                  # Иконка каралтысы
├── requirements.txt         # Python бәйләнешләре
├── docs/                    # Күптелле документлар
│   └── README_*.md
├── i18n/                    # Тәрҗемә файллары
│   └── *.json
├── libs/
│   └── lhm/
│       └── lib/net472/      # LibreHardwareMonitorLib DLL
└── dist/
    └── ChipPeek.exe         # Төзелгән җибәрүче файл
```

## Конфигурация файлы

`config.json` файлы EXE белән бер үк каталогта саклана, барлык үзгәртелә торган көйләүләрне үз эченә ала. Көйләүләр үзгәртелгәндә автоматик рәвештә саклана.
