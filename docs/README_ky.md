## Өзгөчөлүктөр

- **Реалдуу убакта көзөмөл**: CPU жыштыгы, CPU температурасы, GPU жыштыгы, GPU температурасы, VRAM колдонуусу, эстутум колдонуусу
- **Эки көрсөтүү режими**: Бурул виджет / Жогорку тилке, бир баскыч менен алмаштыруу
- **Көрсөтүүнү ыңгайлоо**: Кайсы көрсөткүчтөрдү көрсөтүү керек экенин эркин тандоо, пайыз/чыныгы маанилерди алмаштыруу
- **Дайыма үстүндө**: Бардык терезелердин үстүндө турат, толук экран оюндарда иштейт
- **Авто өлчөм**: Терезе туурасы мазмунга ылайык автоматтык түрдө ыңгайланат
- **Ыңгайлаштырылуучу стиль**: Терезе тунуктугу, фон тунуктугу, тамга өлчөмү баары ыңгайлаштырылат
- **Көп тилдүү колдоо**: 20+ тил, системанын тилин авто аныктайт
- **Ыңгайлуу башкаруу**: Сол сүйрөө менен жылдыруу, оң баскыч менюсу менен тез жөндөөлөр, экран кырларына авто жабышуу
- **Үлгү алуу жөндөөсү**: 200ms - 3000ms ыңгайлаштырылуучу
- **Авто баштоо**: Windows системасына киргенде автоматтык түрдө башталат
- **Аз ресурс колдонуу**: Аркада эң аз ресурс колдонот

## Тез баштоо

### Түз колдонуу

`ChipPeek.exe` жүктөп алып, эки жолу басыңыз (CPU температурасын жана так жыштыкты окуу үчүн автоматтык түрдө администратор укуктарын сурайт).

### Булактан иштетүү

```bash
# Репозиторийди клондоңуз
git clone https://github.com/Glimmer114514/ChipPeek.git
cd ChipPeek

# Көзкарандылыктарды орнотуңуз
pip install -r requirements.txt

# Иштетиңиз
python main.py
```

## Системалык талаптар

- Windows 10 / Windows 11
- Администратор укуктары (сунушталат), болбосо CPU температурасы жана так жыштык жеткиликтүү болбошу мүмкүн
- .NET Framework 4.7.2 же жогору (LibreHardwareMonitor үчүн керек)

## EXE катары куруу

```bash
pip install pyinstaller
pyinstaller --noconfirm --onefile --windowed --name "ChipPeek" --icon "app.ico" --manifest "admin.manifest" --add-data "libs;libs" --add-data "app.ico;." --hidden-import "clr" --hidden-import "pynvml" --hidden-import "win32gui" --hidden-import "win32con" main.py
```

Куруудан кийин, EXE файлы `dist/ChipPeek.exe` жеринде жайгашат.

## Колдонуу

### Негизги амалдар

| Амал | Сүрөттөмө |
|------|------|
| Сол сүйрөө | Терезенин жайгашуусун жылдыруу |
| Оң баскыч | Менюну ачуу (режим алмаштыруу / жөндөөлөр / чыгуу) |
| Авто жабышуу | Экран кырларынын 20px ичинде автоматтык жабышуу |

### Көрсөтүү режимдери

- **Бурул виджет**: Бардык көрсөткүчтөр вертикалдуу жайгашат, экрандын каалаган бурчунда жайгаштырыла алат
- **Жогорку тилке**: Бардык көрсөткүчтөр горизонталдуу жайгашат, демейки боюнча экрандын жогорку ортосунда

### Жөндөөлөр

- **Көрсөтүү режими**: Бурул виджет / Жогорку тилке
- **Виджеттин абалы**: Төмөн оң / Төмөн сол / Жогорку оң / Жогорку сол
- **Терезенин тунуктугу**: 30% - 100% жалпы терезенин тунуктугу
- **Фондун тунуктугу**: 0% - 100% фон тунук (текст так бойдон калат)
- **Үлгү алуу аралыгы**: 200ms - 3000ms маалымат жаңылоо ылдамдыгы
- **Тамга өлчөмү**: 8 - 20 чекит тамга
- **Тил**: 20+ тил, системанын тилин авто аныктайт
- **Көрсөтүү көрсөткүчтөрү**: Ар бир көрсөткүчтү өзүнчө күйгүзүп/өчүрүүгө болот
- **Көрсөтүү форматы**: VRAM / Эстутум пайыз же чыныгы маанилер арасында алмаштырыла алат
- **Авто баштоо**: Windows системасына киргенде автоматтык түрдө иштөө

## Технологиялык стек

- **GUI**：Tkinter
- **Катуу жабдык маалыматы**：LibreHardwareMonitorLib (pythonnet аркылуу), psutil, pynvml (NVIDIA GPU үчүн)
- **Системалык интеграция**：pywin32 (терезе үстүндө, авто баштоо реестри)
- **Упаковка**：PyInstaller

## Долбоордун түзүмү

```
ChipPeek/
├── main.py                  # Кируу чекити
├── monitor_window.py        # Терезе UI жана өз ара аракет логикасы
├── hardware_monitor.py      # Катуу жабдык маалымат чогултуу
├── config.py                # Жөндөөлөрдү башкаруу
├── i18n.py                  # Эл аралыктык
├── generate_icon.py         # Белги түзүү скрипти
├── admin.manifest           # UAC администратор укуктарынын манифести
├── app.ico                  # Тиркеменин белгиси
├── app.png                  # Белгинин алдын ала көрүүсү
├── requirements.txt         # Python көзкарандылыктары
├── docs/                    # Көп тилдүү документтер
│   └── README_*.md
├── i18n/                    # Котормо файлдары
│   └── *.json
├── libs/
│   └── lhm/
│       └── lib/net472/      # LibreHardwareMonitorLib DLL
└── dist/
    └── ChipPeek.exe         # Компиляцияланган аткаруучу
```

## Жөндөө файлы

`config.json` файлы EXE менен бир каталогдо сакталат, бардык ыңгайлаштырылуучу жөндөөлөрдү камтыйт. Жөндөөлөр өзгөргөндө автоматтык түрдө сакталат.
