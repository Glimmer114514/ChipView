# ChipPeek

Windows ਲਈ ਇੱਕ ਹਲਕਾ ਭਾਰ ਦਾ ਫਲੋਟਿੰਗ ਹਾਰਡਵੇਅਰ ਮਾਨੀਟਰਿੰਗ ਵਿਜੇਟ. CPU/GPU ਫ੍ਰੀਕੁਐਂਸੀ, ਤਾਪਮਾਨ, VRAM ਅਤੇ ਮੈਮੋਰੀ ਵਰਤੋਂ ਦੀ ਰੀਅਲ-ਟਾਈਮ ਮਾਨੀਟਰਿੰਗ, ਫੁਲ-ਸਕ੍ਰੀਨ ਐਪਸ ਸਮੇਤ ਹਮੇਸ਼ਾ ਚੋਟੀ ਤੇ ਰਹਿੰਦਾ ਹੈ.

## ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ

- **ਰੀਅਲ-ਟਾਈਮ ਮਾਨੀਟਰਿੰਗ**: CPU ਫ੍ਰੀਕੁਐਂਸੀ, CPU ਤਾਪਮਾਨ, GPU ਫ੍ਰੀਕੁਐਂਸੀ, GPU ਤਾਪਮਾਨ, VRAM ਵਰਤੋਂ, ਮੈਮੋਰੀ ਵਰਤੋਂ
- **ਦੋ ਡਿਸਪਲੇ ਮੋਡ**: ਕੋਰਨਰ ਵਿਜੇਟ / ਟੌਪ ਬਾਰ, ਇੱਕ ਕਲਿਕ ਨਾਲ ਸਵਿੱਚ ਕਰੋ
- **ਅਨੁਕੂਲਿਤ ਡਿਸਪਲੇ**: ਕਿਹੜੇ ਮੈਟ੍ਰਿਕਸ ਦਿਖਾਉਣੇ ਹਨ ਇਹ ਸੁਤੰਤਰਤਾ ਨਾਲ ਚੁਣੋ, ਪ੍ਰਤੀਸ਼ਤ/ਅਸਲ ਮੁੱਲਾਂ ਵਿਚਕਾਰ ਸਵਿੱਚ ਕਰੋ
- **ਹਮੇਸ਼ਾ ਚੋਟੀ ਤੇ**: ਸਾਰੀਆਂ ਵਿੰਡੋਜ਼ ਦੇ ਉੱਪਰ ਰਹਿੰਦਾ ਹੈ, ਫੁਲ-ਸਕ੍ਰੀਨ ਗੇਮਾਂ ਵਿੱਚ ਕੰਮ ਕਰਦਾ ਹੈ
- **ਆਟੋਮੈਟਿਕ ਆਕਾਰ**: ਸਮੱਗਰੀ ਦੇ ਆਧਾਰ ਤੇ ਵਿੰਡੋ ਚੌੜਾਈ ਆਪਣੇ ਆਪ ਅਨੁਕੂਲਿਤ ਹੋ ਜਾਂਦੀ ਹੈ
- **ਵਿਵਸਥਿਤ ਸਟਾਈਲ**: ਵਿੰਡੋ ਪਾਰਦਰਸ਼ਿਤਾ, ਪਿੱਠਭੂਮੀ ਪਾਰਦਰਸ਼ਿਤਾ, ਫੌਂਟ ਆਕਾਰ - ਸਭ ਕੁਝ ਵਿਵਸਥਿਤ
- **ਬਹੁਭਾਸ਼ਾਈ ਸਮਰਥਨ**: 20+ ਭਾਸ਼ਾਵਾਂ, ਆਪਣੇ ਆਪ ਸਿਸਟਮ ਭਾਸ਼ਾ ਦਾ ਪਤਾ ਲਗਾਉਂਦੀ ਹੈ
- **ਆਸਾਨ ਓਪਰੇਸ਼ਨ**: ਖੱਬੇ ਕਲਿਕ ਨਾਲ ਘਸੀਟ ਕੇ ਹਿਲਾਓ, ਸੱਜੇ ਕਲਿਕ ਮੇਨੂ ਤੇਜ਼ ਸੈਟਿੰਗਾਂ ਲਈ, ਸਕ੍ਰੀਨ ਕੰਢਿਆਂ ਤੇ ਆਟੋਮੈਟਿਕ ਸਨੈਪ
- **ਸੈਂਪਲਿੰਗ ਫ੍ਰੀਕੁਐਂਸੀ ਵਿਵਸਥਾ**: 200ms - 3000ms ਵਿਵਸਥਿਤ
- **ਆਟੋ ਸਟਾਰਟ**: Windows ਵਿੱਚ ਲੌਗਇਨ ਤੇ ਆਪਣੇ ਆਪ ਸ਼ੁਰੂ ਹੋ ਜਾਂਦਾ ਹੈ
- **ਘੱਟ ਸਰੋਤ ਵਰਤੋਂ**: ਪਿੱਠਭੂਮੀ ਵਿੱਚ ਘੱਟੋ ਘੱਟ ਥਾਂ

## ਤੇਜ਼ ਸ਼ੁਰੂਆਤ

### ਸਿੱਧੀ ਵਰਤੋਂ

`ChipPeek.exe` ਡਾਊਨਲੋਡ ਕਰੋ ਅਤੇ ਚਲਾਉਣ ਲਈ ਡਬਲ-ਕਲਿਕ ਕਰੋ (CPU ਤਾਪਮਾਨ ਅਤੇ ਸਹੀ ਫ੍ਰੀਕੁਐਂਸੀ ਪੜ੍ਹਨ ਲਈ ਆਪਣੇ ਆਪ ਪ੍ਰਸ਼ਾਸਕ ਅਧਿਕਾਰ ਮੰਗੇਗਾ).

### ਸਰੋਤ ਤੋਂ ਚਲਾਓ

```bash
# ਰਿਪੋਜ਼ਟਰੀ ਕਲੋਨ ਕਰੋ
git clone https://github.com/Glimmer114514/ChipPeek.git
cd ChipPeek

# ਨਿਰਭਰਤਾਵਾਂ ਸਥਾਪਿਤ ਕਰੋ
pip install -r requirements.txt

# ਚਲਾਓ
python main.py
```

## ਸਿਸਟਮ ਲੋੜਾਂ

- Windows 10 / Windows 11
- ਪ੍ਰਸ਼ਾਸਕ ਅਧਿਕਾਰ (ਸਿਫਾਰਸ਼ੀ), ਨਹੀਂ ਤਾਂ CPU ਤਾਪਮਾਨ ਅਤੇ ਸਹੀ ਫ੍ਰੀਕੁਐਂਸੀ ਉਪਲਬਧ ਨਹੀਂ ਹੋ ਸਕਦੀ
- .NET Framework 4.7.2 ਜਾਂ ਉੱਚ (LibreHardwareMonitor ਲਈ ਲੋੜੀਂਦਾ)

## EXE ਵਜੋਂ ਬਿਲਡ ਕਰੋ

```bash
pip install pyinstaller
pyinstaller --noconfirm --onefile --windowed --name "ChipPeek" --icon "app.ico" --manifest "admin.manifest" --add-data "libs;libs" --add-data "app.ico;." --hidden-import "clr" --hidden-import "pynvml" --hidden-import "win32gui" --hidden-import "win32con" main.py
```

ਬਿਲਡ ਕਰਨ ਤੋਂ ਬਾਅਦ, EXE ਫਾਈਲ `dist/ChipPeek.exe` ਵਿੱਚ ਹੈ.

## ਵਰਤੋਂ

### ਬੁਨਿਆਦੀ ਓਪਰੇਸ਼ਨਾਂ

| ਓਪਰੇਸ਼ਨ | ਵੇਰਵਾ |
|------|------|
| ਖੱਬਾ ਕਲਿਕ ਘਸੀਟਣਾ | ਵਿੰਡੋ ਸਥਿਤੀ ਹਿਲਾਓ |
| ਸੱਜਾ ਕਲਿਕ | ਮੇਨੂ ਖੋਲ੍ਹੋ (ਮੋਡ ਸਵਿੱਚ ਕਰੋ / ਸੈਟਿੰਗਾਂ / ਬਾਹਰ ਨਿਕਲੋ) |
| ਆਟੋ ਸਨੈਪ | ਸਕ੍ਰੀਨ ਕੰਢਿਆਂ ਤੋਂ 20px ਅੰਦਰ ਆਪਣੇ ਆਪ ਸਨੈਪ ਹੋ ਜਾਂਦਾ ਹੈ |

### ਡਿਸਪਲੇ ਮੋਡ

- **ਕੋਰਨਰ ਵਿਜੇਟ**: ਸਾਰੇ ਮੈਟ੍ਰਿਕਸ ਲੰਬਕ੍ਰਮ ਵਿਵਸਥਾ ਵਿੱਚ ਸਜਾਏ ਗਏ ਹਨ, ਸਕ੍ਰੀਨ ਦੇ ਕਿਸੇ ਵੀ ਕੋਨੇ ਵਿੱਚ ਰੱਖੇ ਜਾ ਸਕਦੇ ਹਨ
- **ਟੌਪ ਬਾਰ**: ਸਾਰੇ ਮੈਟ੍ਰਿਕਸ ਲੇਟਵੀਂ ਵਿਵਸਥਾ ਵਿੱਚ ਸਜਾਏ ਗਏ ਹਨ, ਡਿਫੌਲਟ ਰੂਪ ਵਿੱਚ ਸਕ੍ਰੀਨ ਦੇ ਸਿਰਲੇਖ ਤੇ ਦਰਮਿਆਨ ਵਿੱਚ ਹਨ

### ਸੈਟਿੰਗਾਂ

- **ਡਿਸਪਲੇ ਮੋਡ**: ਕੋਰਨਰ ਵਿਜੇਟ / ਟੌਪ ਬਾਰ
- **ਵਿਜੇਟ ਸਥਿਤੀ**: ਹੇਠਾਂ ਸੱਜਾ / ਹੇਠਾਂ ਖੱਬਾ / ਟੌਪ ਸੱਜਾ / ਟੌਪ ਖੱਬਾ
- **ਵਿੰਡੋ ਪਾਰਦਰਸ਼ਿਤਾ**: 30% - 100% ਕੁੱਲ ਵਿੰਡੋ ਪਾਰਦਰਸ਼ਿਤਾ
- **ਪਿੱਠਭੂਮੀ ਪਾਰਦਰਸ਼ਿਤਾ**: 0% - 100% ਪਿੱਠਭੂਮੀ ਪਾਰਦਰਸ਼ਿਤਾ (ਟੈਕਸਟ ਸਪਸ਼ਟ ਰਹਿੰਦਾ ਹੈ)
- **ਸੈਂਪਲਿੰਗ ਅੰਤਰਾਲ**: 200ms - 3000ms ਡੇਟਾ ਰਿਫਰੈਸ਼ ਦਰ
- **ਫੌਂਟ ਆਕਾਰ**: 8 - 20 ਬਿੰਦੂ
- **ਭਾਸ਼ਾ**: 20+ ਭਾਸ਼ਾਵਾਂ, ਆਪਣੇ ਆਪ ਸਿਸਟਮ ਭਾਸ਼ਾ ਦਾ ਪਤਾ ਲਗਾਉਂਦੀ ਹੈ
- **ਡਿਸਪਲੇ ਮੈਟ੍ਰਿਕਸ**: ਹਰੇਕ ਮੈਟ੍ਰਿਕ ਨੂੰ ਸੁਤੰਤਰ ਰੂਪ ਨਾਲ ਚਾਲੂ/ਬੰਦ ਕਰੋ
- **ਡਿਸਪਲੇ ਫਾਰਮੈਟ**: VRAM / ਮੈਮੋਰੀ ਪ੍ਰਤੀਸ਼ਤ ਜਾਂ ਅਸਲ ਮੁੱਲਾਂ ਵਿਚਕਾਰ ਸਵਿੱਚ ਕਰ ਸਕਦਾ ਹੈ
- **ਆਟੋ ਸਟਾਰਟ**: Windows ਵਿੱਚ ਲੌਗਇਨ ਤੇ ਆਪਣੇ ਆਪ ਚਲਾਓ

## ਟੈਕ ਸਟੈਕ

- **GUI**：Tkinter
- **ਹਾਰਡਵੇਅਰ ਡੇਟਾ**：LibreHardwareMonitorLib (pythonnet ਰਾਹੀਂ), psutil, pynvml (NVIDIA GPU ਲਈ ਬੈਕਅੱਪ)
- **ਸਿਸਟਮ ਏਕੀਕਰਣ**：pywin32 (ਵਿੰਡੋ ਹਮੇਸ਼ਾ ਚੋਟੀ ਤੇ, ਆਟੋ ਸਟਾਰਟ ਰਜਿਸਟਰੀ)
- **ਪੈਕੇਜਿੰਗ**：PyInstaller

## ਕੌਂਫਿਗਰੇਸ਼ਨ ਫਾਈਲ

`config.json` ਫਾਈਲ EXE ਦੇ ਉਸੇ ਡਾਇਰੈਕਟਰੀ ਵਿੱਚ ਸੇਵ ਹੁੰਦੀ ਹੈ ਅਤੇ ਸਾਰੀਆਂ ਵਿਵਸਥਿਤ ਸੈਟਿੰਗਾਂ ਹੁੰਦੀਆਂ ਹਨ. ਸੈਟਿੰਗਾਂ ਬਦਲਣ ਤੇ ਆਪਣੇ ਆਪ ਸੇਵ ਹੋ ਜਾਂਦੀਆਂ ਹਨ.
