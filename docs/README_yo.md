## Àwọn Àbájáde

- **Ṣàyẹ̀wò ní àkókò gidi**: Ìgbohùn CPU, ìwọ̀n CPU, ìgbohùn GPU, ìwọ̀n GPU, ìlò VRAM, ìlò ìrántí
- **Ọ̀nà ìṣàfihàn méjì**: Widget àgbáǹtá / Ilẹ̀ òkè, yípadà pẹ̀lú títẹ ẹ̀yìn kan
- **Ìṣàfihàn tó lè yípadà**: Lààyè láti yan àwọn iye tí ó fẹ́ kí ó hàn, yípadà láàrín iye/orin iye gan an
- **Dúró lókàn kíkan**: Ó ń dúró lókàn gbogbo fèrèsé, ó sì ń ṣiṣẹ́ nínú àwọn eré ìhà kíkún
- **Ìwọ̀n ara rẹ̀**: Fífè fèrèsé máa ń yípadà ara rẹ̀ láti bá àkóónù mu
- **Àwọn àrídájú tó lè yípadà**: Ìfòyemọ́ fèrèsé, ìfòyemọ́ ìpìlẹ̀, ìwọ̀n fontì gbogbo rẹ̀ lè yípadà
- **Àtìlẹyìn èdè púpọ̀**: Èdè 20+, ó ń wá èdè ètò ní ọ̀nà ara rẹ̀
- **Iṣẹ́ rọ̀ọ̀**: Fá àyìká láti gbe, tẹ ọwọ́ ọ̀tún fún atojọ àtúnṣe lólé, Ìfilé fúnra rẹ̀ sí etí ìhà
- **Àkókò ìyípadà tó lè yípadà**: 200ms - 3000ms lè yípadà
- **Bẹ̀rẹ̀ fúnra rẹ̀**: Bẹ̀rẹ̀ ní ọ̀nà ara rẹ̀ nígbà tí o bá wọlé sí Windows
- **Ìlò ààyè kékèké**: Kékèké ní ìhà ẹ̀yìn

## Bẹ̀rẹ̀ Lólé

### Lò Ó ní Àkúrọ̀

Gbà `ChipPeek.exe` sókè kí o sì tẹ e lẹ́ẹ̀mejì láti rọ̀ṣọ (ó máa béèrè fún àṣẹ oníṣẹ̀ àkóso láti ka ìwọ̀n CPU àti ìgbohùn tó yẹ).

### Rọ̀ṣọ Láti Inú Orísun

```bash
# Kó nù ìgbìmọ̀
git clone https://github.com/Glimmer114514/ChipPeek.git
cd ChipPeek

# Fifi àwọn ohun elè sì
pip install -r requirements.txt

# Rọ̀ṣọ
python main.py
```

## Àwọn Ìbéèrè Ètò

- Windows 10 / Windows 11
- Àṣẹ oníṣẹ̀ àkóso (a ṣàkíyèsí), bibẹ́ẹ̀kọ́ ìwọ̀n CPU àti ìgbohùn tó yẹ lè má wà
- .NET Framework 4.7.2 tàbí jù bẹ́ẹ̀ lọ (LibreHardwareMonitor nílò rẹ̀)

## Kó gẹ́gẹ́ bí EXE

```bash
pip install pyinstaller
pyinstaller --noconfirm --onefile --windowed --name "ChipPeek" --icon "app.ico" --manifest "admin.manifest" --add-data "libs;libs" --add-data "app.ico;." --hidden-import "clr" --hidden-import "pynvml" --hidden-import "win32gui" --hidden-import "win32con" main.py
```

Lẹ́yìn tí a bá kọ tán, faịlì EXE wà ní `dist/ChipPeek.exe`.

## Bí a ṣe ń Lo

### Àwọn Iṣẹ́ Àkọ́kọ́

| Iṣẹ́ | Àpèjúwe |
|------|------|
| Fá àyìká | Gbe ibi fèrèsé |
| Tẹ ọ̀tún | Ṣí atojọ (yí ọ̀nà padà / àtúnṣe / jáde) |
| Ìfilé fúnra rẹ̀ | Ó máa ń filé fúnra rẹ̀ sínú 20px ti etí ìhà |

### Àwọn Ọ̀nà Ìṣàfihàn

- **Widget àgbáǹtá**: Gbogbo àwọn iye wà ní gígun, a lè gbe sínú ẹ̀yínkákan ti ìhà
- **Ilẹ̀ òkè**: Gbogbo àwọn iye wà ní ìífẹ̀rẹsẹ̀, ní àárín ní òkè ìhà ní ọ̀nà ìbẹ̀rẹ̀

### Àtúnṣe

- **Ọ̀nà ìṣàfihàn**: Widget àgbáǹtá / Ilẹ̀ òkè
- **Ibi widget**: Ìsàlẹ̀ òṣì / Ìsàlẹ̀ òsì / Òkè òṣì / Òkè òsì
- **Ìfòyemọ́ fèrèsé**: 30% - 100% ìfòyemọ́ fèrèsé gbogbo rẹ̀
- **Ìfòyemọ́ ìpìlẹ̀**: 0% - 100% ìpìlẹ̀ kékèké (ọ̀rọ̀ dúró láyọ)
- **Àkókò ìyípadà**: 200ms - 3000ms ìyípadà ìmúdàgba data
- **Ìwọ̀n fontì**: 8 - 20 ìwọ̀n fontì
- **Èdè**: Èdè 20+, ó ń wá èdè ètò ní ọ̀nà ara rẹ̀
- **Ìṣàfihàn iye**: Yí ọ̀kọ̀kan kọ̀ọ̀kan padà ní ọ̀nà ara rẹ̀
- **Ọ̀nà ìṣàfihàn**: VRAM / Ìrántí lè yípadà láàrín iye tàbí iye gan an
- **Bẹ̀rẹ̀ fúnra rẹ̀**: Rọ̀ṣọ ní ọ̀nà ara rẹ̀ nígbà tí o bá wọlé sí Windows

## Àwọn Ohun Ìṣe

- **GUI**：Tkinter
- **Data ohun èlò**：LibreHardwareMonitorLib (nipasẹ pythonnet), psutil, pynvml (àkóbí NVIDIA GPU)
- **Ìjápọ̀ ètò**：pywin32 (fèrèsé lókàn, ìforúkọsílẹ̀ bẹ̀rẹ̀ fúnra rẹ̀)
- **Ìkójọpọ̀**：PyInstaller

## Faịlì Àtúnṣe

Faịlì `config.json` à ń fi pamọ́ sínú àpò kanna tí EXE wà, ó sì ní gbogbo àwọn àtúnṣe tó lè yípadà. A máa ń fi àwọn àtúnṣe pamọ́ ní ọ̀nà ara wọn nígbà tí a bá yí wọ́n padà.
