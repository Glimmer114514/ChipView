## Мөмкинлектәр

- **Реаль ваҡытта күҙәтеү**: CPU йышлығы, CPU температураһы, GPU йышлығы, GPU температураһы, VRAM ҡулланыуы, хәтер ҡулланыуы
- **Ике күрһәтеү рәүеше**: Мөйөш виджеты / Өҫкө таҡта, бер баҫыу менән алмаштырыу
- **Үҙгәртелә торған күрһәтеү**: күрһәтелә торған метриктарҙы ирекле һайлау, процент/ысын ҡиммәт алмаштырыу
- **Һәр ваҡыт өҫтә**: барлыҡ тәрәзәләр өҫтөндә тора, тулы экранлы уйындарҙа эшләй
- **Авто-ҙурлыҡ**: тәрәзә киңлеге эстәлеккә ҡарап автоматик рәүештә үҙгәрә
- **Үҙгәртелә торған стиль**: тәрәзә күренеше, фон үтә күренеүсәнлеге, хәреф ҙурлығы үҙгәртелә ала
- **Күптелле ярҙам**: 20+ тел, система телен автоматик рәүештә асыҡлау
- **Уңайлы ҡулланыу**: һул һөйрәлеү менән күсереү, уң баҫыу менюһы менән тиҙ көйләүҙәр, экран ситтәренә автоматик йәбешеү
- **Конфигурациялана торған үлсәү**: 200ms - 3000ms үҙгәртелә ала
- **Автоматик башланыу**: Windows-ҡа ингәндә автоматик рәүештә башлана
- **Аҙ ресурслы ҡулланыу**: артҡы планда минималь эҙ

## Тиҙ башланыу

### Туры ҡулланыу

`ChipPeek.exe` файлын йөкләгеҙ һәм ике тапҡыр баҫып эшләтеп ебәрегеҙ (CPU температураһы һәм теүәл йышлыҡ уҡып алыу өсөн автоматик рәүештә администатор хоҡуҡтарын һораясаҡ).

### Сығанаҡтан эшләтеп ебәреү

```bash
# Репозиторийҙы клонлау
git clone https://github.com/Glimmer114514/ChipPeek.git
cd ChipPeek

# Бәйләнештәрҙе урынлаштырыу
pip install -r requirements.txt

# Эшләтеп ебәреү
python main.py
```

## Система талаптары

- Windows 10 / Windows 11
- Администратор хоҡуҡтары (тәҡдим ителә), юғиһә CPU температураһы һәм теүәл йышлыҡ булмауы ихтимал
- .NET Framework 4.7.2 йәки юғарыраҡ (LibreHardwareMonitor тарафынан талап ителә)

## EXE итеп төҙөү

```bash
pip install pyinstaller
pyinstaller --noconfirm --onefile --windowed --name "ChipPeek" --icon "app.ico" --manifest "admin.manifest" --add-data "libs;libs" --add-data "app.ico;." --hidden-import "clr" --hidden-import "pynvml" --hidden-import "win32gui" --hidden-import "win32con" main.py
```

Төҙөлгәндән һуң, EXE файлы `dist/ChipPeek.exe` адресында урынлаша.

## Ҡулланыу

### Төп эш-ғәмәлдәр

| Эш-ғәмәл | Тасуирлама |
|------|------|
| Һул һөйрәлеү | Тәрәзәне күсереү |
| Уң баҫыу | Меню асыу (рәүеште алмаштырыу / көйләүҙәр / сығыу) |
| Автоматик йәбешеү | Экран ситтәренә 20px эсендә автоматик йәбешә |

### Күрһәтеү рәүештәре

- **Мөйөш виджеты**: барлыҡ метриктар вертикаль рәүештә урынлаштырылған, экрандың теләһә ниндәй мөйөшөнә ҡуйырға мөмкин
- **Өҫкө таҡта**: барлыҡ метриктар горизонталь рәүештә урынлаштырылған, әйткәндәй, экран өҫтөндә үҙәкләштерелгән

### Көйләүҙәр

- **Күрһәтеү рәүеше**: Мөйөш виджеты / Өҫкө таҡта
- **Виджет урыны**: Аҫта уңда / Аҫта һулда / Өҫтә уңда / Өҫтә һулда
- **Тәрәзә күренеше**: дөйөм тәрәзә үтә күренеүсәнлеге 30% - 100%
- **Фон үтә күренеүсәнлеге**: фон үтә күренеүсәнлеге 0% - 100% (текст асыҡ ҡала)
- **Үлсәү арауығы**: мәғлүмәт яңыртыу йышлығы 200ms - 3000ms
- **Хәреф ҙурлығы**: 8 - 20 нөктә хәреф
- **Тел**: 20+ тел, система телен автоматик рәүештә асыҡлау
- **Күрһәтелә торған метриктар**: һәр метриканы айырым үҙгәртеү
- **Күрһәтеү форматы**: VRAM / Хәтер процент менән йә ысын ҡиммәт араһында алмаштырырға мөмкин
- **Автоматик башланыу**: Windows-ҡа ингәндә автоматик рәүештә эшләтеп ебәреү

## Технологик стек

- **График интерфейс**：Tkinter
- **Ҡорамал мәғлүмәте**：LibreHardwareMonitorLib (pythonnet аша), psutil, pynvml (NVIDIA GPU запас сисеше)
- **Система интеграцияһы**：pywin32 (тәрәзә өҫтә, автостат реестры)
- **Урынлаштырыу**：PyInstaller

## Проект структураһы

```
ChipPeek/
├── main.py                  # Инеш нөктәһе
├── monitor_window.py        # Тәрәзә UI һәм интерактив логика
├── hardware_monitor.py      # Ҡорамал мәғлүмәте йыйыу
├── config.py                # Конфигурация менән идара итеү
├── i18n.py                  # Халыҡ-аралаштырыу
├── generate_icon.py         # Иконка генерациялау скрипты
├── admin.manifest           # UAC администратор хоҡуғы манифесты
├── app.ico                  # Ҡушымта иконкаһы
├── app.png                  # Иконка ҡаралтыһы
├── requirements.txt         # Python бәйләнештәре
├── docs/                    # Күптелле документтар
│   └── README_*.md
├── i18n/                    # Тәржемә файлдары
│   └── *.json
├── libs/
│   └── lhm/

```

## Конфигурация файлы

`config.json` файлы EXE менән бер үк каталогта һаҡлана, бөтә үҙгәртелә торған көйләүҙәрҙе үҙ эсенә ала. Көйләүҙәр үҙгәртелгәндә автоматик рәүештә һаҡлана.
