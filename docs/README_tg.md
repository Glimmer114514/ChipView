## Хусусиятҳо

- **Назорати realtime**: Басомади CPU, ҳарорати CPU, басомади GPU, ҳарорати GPU, истифодаи VRAM, истифодаи ҳофиза
- **Ду ҳолати намоиш**: Виҷети кунҷ / Навори боло, бо як клик иваз кардан
- **Намоиши дудконӣ**: Интихоби озоди нишондиҳандаҳо, иваз кардани фоиз/арзишҳои воқеӣ
- **Доим боло**: Дар болои ҳама тирезаҳо мемонад, дар бозиҳои fullscreen кор мекунад
- **Андозаи худкор**: Барои мувофиқ кардан ба мӯҳтаво паҳнои тиреза худкор танзим мешавад
- **Услуби танзимшаванда**: Шаффофии тиреза, шаффофии пасзамина, андозаи ҳуруф ҳамааш танзимшаванда
- **Дастгирии чандзабона**: 20+ забон, забони системаро худкор муайян мекунад
- **Амали осон**: Бо кашидани чап ҳаракат, мению танзимоти зуд бо клики рост, худчаспидан ба кунҷҳои экран
- **Танзими намунагирӣ**: 200ms - 3000ms танзимшаванда
- **Оғози худкор**: Ҳангоми воридшавии Windows худкор оғоз мешавад
- **Истифодаи камтарин захира**: Дар пасзамина камтарин истифодаи захира

## Оғози зуд

### Истифодаи мустақим

`ChipPeek.exe`-ро зер кашӣ карда, ду бор клик кунед (барои хондани ҳарорати CPU ва басомади дақиқ худкор ҳуқуқи идоракунандаро дархост мекунад).

### Аз сарчашма иҷро кардан

```bash
# Клон кардани репозитория
git clone https://github.com/Glimmer114514/ChipPeek.git
cd ChipPeek

# Насб кардани вобастагиҳо
pip install -r requirements.txt

# Иҷро кардан
python main.py
```

## Талаботи системавӣ

- Windows 10 / Windows 11
- Ҳуқуқҳои идоракунанда (тавсия мешавад), вагарна ҳарорати CPU ва басомади дақиқ дастрас намешавад
- .NET Framework 4.7.2 ё болотар (барои LibreHardwareMonitor лозим аст)

## Ҳамчун EXE сохтан

```bash
pip install pyinstaller
pyinstaller --noconfirm --onefile --windowed --name "ChipPeek" --icon "app.ico" --manifest "admin.manifest" --add-data "libs;libs" --add-data "app.ico;." --hidden-import "clr" --hidden-import "pynvml" --hidden-import "win32gui" --hidden-import "win32con" main.py
```

Пас аз сохтан, файли EXE дар `dist/ChipPeek.exe` ҷойгир аст.

## Истифода

### Амалҳои асосӣ

| Амал | Тасвир |
|------|------|
| Кашидани чап | Ҷойивазкунии тиреза |
| Клики рост | Кушодани меню (иваз кардани ҳолат / танзимот / баромад) |
| Худчаспидан | Дар 20px аз кунҷҳои экран худкор часпидан |

### Ҳолатҳои намоиш

- **Виҷети кунҷ**: Ҳамаи нишондиҳандаҳо ба таври амудӣ ҷойгир мешаванд, дар ҳар кунҷи экран ҷойгир карда мешавад
- **Навори боло**: Ҳамаи нишондиҳандаҳо ба таври уфуқӣ ҷойгир мешаванд, бо нобаёнӣ дар маркази болои экран

### Танзимот

- **Ҳолати намоиш**: Виҷети кунҷ / Навори боло
- **Мавқеи виҷет**: Поёни рост / Поёни чап / Болои рост / Болои чап
- **Шаффофии тиреза**: 30% - 100% шаффофии умумии тиреза
- **Шаффофии пасзамина**: 0% - 100% шаффофии пасзамина (матт боқӣ мемонад)
- **Фосилаи намунагирӣ**: 200ms - 3000ms суръати навсозии маълумот
- **Андозаи ҳуруф**: 8 - 20 нуқта ҳуруф
- **Забон**: 20+ забон, забони системаро худкор муайян мекунад
- **Нишондиҳандаҳои намоиш**: Ҳар нишондиҳандаро алоҳида фаъол/ғайрифаъол кунед
- **Формати намоиш**: VRAM / Ҳофиза метавонад байни фоиз ё арзишҳои воқеӣ иваз шавад
- **Оғози худкор**: Ҳангоми воридшавии Windows худкор иҷро шавад

## Анбори технология

- **GUI**：Tkinter
- **Маълумоти сахтафзор**：LibreHardwareMonitorLib (тавассути pythonnet), psutil, pynvml (барои NVIDIA GPU)
- **Интегратсияи системавӣ**：pywin32 (тирезаи боло, оғози худкори реестр)
- **Бастабандӣ**：PyInstaller

```
├── config.py                # Идоракунии танзимот
├── i18n.py                  # Байналмилалӣ кардан
├── generate_icon.py         # Скрипти сохтани айкон
├── admin.manifest           # Манифести ҳуқуқи UAC идоракунанда
├── app.ico                  # Айкон барнома
├── app.png                  # Пешнамоиши айкон
├── requirements.txt         # Вобастагиҳои Python
├── docs/                    # Ҳуҷҷатҳои бисёрзабона
│   └── README_*.md
├── i18n/                    # Файлҳои тарҷума
│   └── *.json
├── libs/

```

## Файли танзимот

Файли `config.json` дар ҳамон луҷаи EXE нигоҳ дошта мешавад, ки ҳамаи танзимотҳои дигаргуншавандаро дар бар мегирад. Танзимотҳо ҳангоми тағйир худкор нигоҳ дошта мешаванд.
