## Taybetmendî

- **Çavdêriya real-time**: Frekansa CPU, germahiya CPU, frekansa GPU, germahiya GPU, bikaranîna VRAM, bikaranîna bîrê
- **Dû mode nîşandanê**: Widget a goşe / Darik a jor, bi yek tikandinê veguherîne
- **Nîşandana ku dikare were sererastkirin**: Hûn dikarin metrîkên ku were nîşandan bijêrin, rêjeyê/nirxa rastîn biguherînin
- **Herdem li jor**: Li ser hemû paceyan dimîne, di lîstikên fullscreen de jî dixebite
- **Mezinahiya xwebixwe**: Firehiya paceyê bi otomatîk li gorî naverokê dixebite
- **Şêwaza ku dikare were sererastkirin**: Opasitiya paceyê, zelaliya paşzemînê, mezinahiya fontê hemû dikare were sererastkirin
- **Piştgiriya pirzimanî**: 20+ ziman, zimanê pergalê xwebixwe nas dike
- **Xebata hêsan**: Kişandina çepê ji bo guhertinê, tikandina rastê ji bo menuya lez, xwebixwe li qeraxên ekranê vekeve
- **Samplinga ku dikare were sererastkirin**: 200ms - 3000ms
- **Xwebixwe dest pê bike**: Gava têkeve Windowsê xwebixwe dest pê dike
- **Bikaranîna çavkaniyên kêm**: Di paşzemînê de şopeke herî kêm

## Destpêkirina Lez

### Bikaranîna Rasterast

`ChipPeek.exe` dakêşin û du caran bitikînin da ku bimeşe (ji bo xwendina germahiya CPU û frekansa rastîn xwebixwe mafê admin daxwaz dike).

### Ji Çavkaniyê Bimeşîne

```bash
# Repository clon bike
git clone https://github.com/Glimmer114514/ChipPeek.git
cd ChipPeek

# Girêdayiyan saz bike
pip install -r requirements.txt

# Bimeşîne
python main.py
```

## Pêdiviyên Pergalê

- Windows 10 / Windows 11
- Mafê administrator (tê pêşniyar kirin), wekî din germahiya CPU û frekansa rastîn dibe ku neyê berdan
- .NET Framework 4.7.2 an bilindtir (ji hêla LibreHardwareMonitor ve tê xwestin)

## Wekî EXE Avabike

```bash
pip install pyinstaller
pyinstaller --noconfirm --onefile --windowed --name "ChipPeek" --icon "app.ico" --manifest "admin.manifest" --add-data "libs;libs" --add-data "app.ico;." --hidden-import "clr" --hidden-import "pynvml" --hidden-import "win32gui" --hidden-import "win32con" main.py
```

Piştî çêkirinê, pelê EXE li `dist/ChipPeek.exe` ye.

## Bikaranîn

### Xebatên Bingehîn

| Çalakî | Terîf |
|------|------|
| Kişandina çepê | Cihê paceyê biguherîne |
| Tikandina rastê | Menuyê veke (mode biguherîne / mîheng / derkeve) |
| Xwebixwe vekevîn | Di nav 20px ji qeraxên ekranê de xwebixwe vekeve |

### Modeyên Nîşandanê

- **Widget a goşe**: Hemû metrîk bi rengekî vekele têne rêzkirin, dikare li her goşeyekî ekranê were danîn
- **Darik a jor**: Hemû metrîk bi rengekî wesa têne rêzkirin, bi default li navenda jor ya ekranê ye

### Mîheng

- **Mode nîşandanê**: Widget a goşe / Darik a jor
- **Cihê widget**: Jêr rast / Jêr çep / Jor rast / Jor çep
- **Opasitiya paceyê**: 30% - 100% opasitiya giştî ya paceyê
- **Zelaliya paşzemînê**: 0% - 100% paşzemîna zelal (nivîs zelal dimîne)
- **Navbera nimûneyan**: 200ms - 3000ms leza nûkirina daneyê
- **Mezinahiya font**: 8 - 20 xalên font
- **Ziman**: 20+ ziman, zimanê pergalê xwebixwe nas dike
- **Metrîkên nîşandanê**: Her metrîkê serbixwe veke/bigire
- **Formata nîşandanê**: VRAM / Bîr dikare di nav rêje an nirxa rastîn de were guhertin
- **Xwebixwe dest pê bike**: Gava têkeve Windowsê xwebixwe bimeşe

## Stacka Teknolojiyê

- **GUI**：Tkinter
- **Daneyên hardware**：LibreHardwareMonitorLib (bi pythonnet re), psutil, pynvml (ji bo NVIDIA GPU yedek)
- **Yekkirina pergalê**：pywin32 (paceya topmost, tomarkirina xwebixwe destpêkirinê)
- **Pakêt kirin**：PyInstaller

## Struktura Projektê

```
ChipPeek/
├── main.py                  # Nuqteya têketinê
├── monitor_window.py        # UI ya paceyê û mantîqaya têkiliyê
├── hardware_monitor.py      # Berhevkirina daneyên hardware
├── config.py                # Birêvebirina mîhengan
├── i18n.py                  # Navneteweyîkirin
├── generate_icon.py         # Skripta çêkirina îkonê
├── admin.manifest           # Manifesta mafê admin a UAC
├── app.ico                  # Îkona sepanê
├── app.png                  # Pêşdîtina îkonê
├── requirements.txt         # Girêdayiyên Python
├── docs/                    # Belgeyên pirzimanî
│   └── README_*.md
├── i18n/                    # Pelên wergerê
│   └── *.json
├── libs/
│   └── lhm/
│       └── lib/net472/      # LibreHardwareMonitorLib DLL
└── dist/
    └── ChipPeek.exe         # Executable a berhevkrî
```

## Pelê Mîhengan

Pelê `config.json` di heman peldankê de wekî EXE tê tomarkirin, hemû mîhengên ku dikare were sererastkirin dihewîne. Mîheng dema ku were guhertin xwebixwe têne tomarkirin.
