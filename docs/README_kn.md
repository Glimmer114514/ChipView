# ChipPeek

Windows ಗಾಗಿ ಈಗಾದರೂ ಹಗುರವಾದ ಫ್ಲೋಟಿಂಗ್ ಹಾರ್ಡ್ವೇರ್ ಮಾನಿಟರಿಂಗ್ ವಿಜೆಟ್. CPU/GPU ಆವರ್ತನ, ತಾಪಮಾನ, VRAM ಮತ್ತು ಮೆಮೊರಿ ಬಳಕೆಯ ನೈಜ ಸಮಯ ಮಾನಿಟರಿಂಗ್, ಪೂರ್ಣ ಪರದೆ ಅಪ್ಲಿಕೇಶನ್‌ಗಳು ಸಹಿತ ಯಾವಾಗಲೂ ಮೇಲೆ ಇರುತ್ತದೆ.

## ವೈಶಿಷ್ಟ್ಯಗಳು

- **ನೈಜ ಸಮಯ ಮಾನಿಟರಿಂಗ್**: CPU ಆವರ್ತನ, CPU ತಾಪಮಾನ, GPU ಆವರ್ತನ, GPU ತಾಪಮಾನ, VRAM ಬಳಕೆ, ಮೆಮೊರಿ ಬಳಕೆ
- **ಎರಡು ಪ್ರದರ್ಶನ ಮೋಡ್‌ಗಳು**: ಕಾರ್ನರ್ ವಿಜೆಟ್ / ಟಾಪ್ ಬಾರ್, ಒಂದು ಕ್ಲಿಕ್‌ನಲ್ಲಿ ಬದಲಾಯಿಸಿ
- **ಕಸ್ಟಮೈಸ್ ಪ್ರದರ್ಶನ**: ಯಾವ ಮೆಟ್ರಿಕ್‌ಗಳನ್ನು ಪ್ರದರ್ಶಿಸಬೇಕು ಎಂದು ಸ್ವತಂತ್ರವಾಗಿ ಆಯ್ಕೆಮಾಡಿ, ಶೇಕಡಾ/ವಾಸ್ತವಿಕ ಮೌಲ್ಯಗಳ ನಡುವೆ ಬದಲಾಯಿಸಿ
- **ಯಾವಾಗಲೂ ಮೇಲೆ**: ಎಲ್ಲಾ ವಿಂಡೋಗಳ ಮೇಲೆ ಇರುತ್ತದೆ, ಪೂರ್ಣ ಪರದೆ ಗೇಮ್‌ಗಳಲ್ಲಿ ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತದೆ
- **ಸ್ವಯಂಚಾಲಿತ ಗಾತ್ರ**: ವಿಷಯದ ಆಧಾರದ ಮೇಲೆ ವಿಂಡೋ ಅಗಲ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಸರಿಹೊಂದಿಸುತ್ತದೆ
- **ಸರಿಹೊಂದಿಸಬಹುದಾದ ಶೈಲಿ**: ವಿಂಡೋ ಪಾರದರ್ಶಕತೆ, ಹಿನ್ನೆಲೆ ಪಾರದರ್ಶಕತೆ, ಫಾಂಟ್ ಗಾತ್ರ - ಎಲ್ಲವೂ ಸರಿಹೊಂದಿಸಬಹುದು
- **ಬಹುಭಾಷೆ ಬೆಂಬಲ**: 20+ ಭಾಷೆಗಳು, ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಸಿಸ್ಟಮ್ ಭಾಷೆಯನ್ನು ಪತ್ತೆಮಾಡುತ್ತದೆ
- **ಸುಲಭ ಕಾರ್ಯಾಚರಣೆ**: ಎಡ ಕ್ಲಿಕ್ ಮಾಡಿ ಎಳೆದು ಸರಿಸು, ಬಲ ಕ್ಲಿಕ್ ಮೆನು ತ್ವರಿತ ಸೆಟ್ಟಿಂಗ್‌ಗಳಿಗಾಗಿ, ಪರದೆ ಅಂಚುಗಳಲ್ಲಿ ಸ್ವಯಂಚಾಲಿತ ಸ್ನ್ಯಾಪ್
- **ಮಾದರಿಕರಣ ಆವರ್ತನ ಸರಿಹೊಂದಿಕೆ**: 200ms - 3000ms ಸರಿಹೊಂದಿಸಬಹುದು
- **ಆಟೋ ಸ್ಟಾರ್ಟ್**: Windows ನಲ್ಲಿ ಲಾಗಿನ್ ಮಾಡಿದಾಗ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಪ್ರಾರಂಭವಾಗುತ್ತದೆ
- **ಕಡಿಮೆ ಸಂಪನ್ಮೂಲ ಬಳಕೆ**: ಹಿನ್ನೆಲೆಯಲ್ಲಿ ಕನಿಷ್ಠ ಸ್ಥಳ

## ತ್ವರಿತ ಪ್ರಾರಂಭ

### ನೇರ ಬಳಕೆ

`ChipPeek.exe` ಅನ್ನು ಡೌನ್ಲೋಡ್ ಮಾಡಿ ಮತ್ತು ಚಲಾಯಿಸಲು ಡಬಲ್-ಕ್ಲಿಕ್ ಮಾಡಿ (CPU ತಾಪಮಾನ ಮತ್ತು ನಿಖರವಾದ ಆವರ್ತನವನ್ನು ಓದಲು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ನಿರ್ವಾಹಕ ಅನುಮತಿಯನ್ನು ವಿನಂತಿಸುತ್ತದೆ).

### ಮೂಲದಿಂದ ಚಲಾಯಿಸಿ

```bash
# ರಿಪೋಜಿಟರಿಯನ್ನು ಕ್ಲೋನ್ ಮಾಡಿ
git clone https://github.com/Glimmer114514/ChipPeek.git
cd ChipPeek

# ಅವಲಂಬನೆಗಳನ್ನು ಸ್ಥಾಪಿಸಿ
pip install -r requirements.txt

# ಚಲಾಯಿಸಿ
python main.py
```

## ಸಿಸ್ಟಮ್ ಅವಶ್ಯಕತೆಗಳು

- Windows 10 / Windows 11
- ನಿರ್ವಾಹಕ ಅನುಮತಿ (ಶಿಫಾರಸು ಮಾಡಲಾಗಿದೆ), ಇಲ್ಲದಿದ್ದರೆ CPU ತಾಪಮಾನ ಮತ್ತು ನಿಖರವಾದ ಆವರ್ತನ ಲಭ್ಯವಿರಬಹುದಿಲ್ಲ
- .NET Framework 4.7.2 ಅಥವಾ ಅದರ ಮೇಲೆ (LibreHardwareMonitor ಗಾಗಿ ಅಗತ್ಯ)

## EXE ಆಗಿ ಬಿಲ್ಡ್ ಮಾಡಿ

```bash
pip install pyinstaller
pyinstaller --noconfirm --onefile --windowed --name "ChipPeek" --icon "app.ico" --manifest "admin.manifest" --add-data "libs;libs" --add-data "app.ico;." --hidden-import "clr" --hidden-import "pynvml" --hidden-import "win32gui" --hidden-import "win32con" main.py
```

ಬಿಲ್ಡ್ ಮಾಡಿದ ನಂತರ, EXE ಫೈಲ್ `dist/ChipPeek.exe` ನಲ್ಲಿ ಇದೆ.

## ಬಳಕೆ

### ಮೂಲ ಕಾರ್ಯಾಚರಣೆಗಳು

| ಕಾರ್ಯಾಚರಣೆ | ವಿವರಣೆ |
|------|------|
| ಎಡ ಕ್ಲಿಕ್ ಎಳೆಯಲು | ವಿಂಡೋ ಸ್ಥಾನವನ್ನು ಸರಿಸಿ |
| ಬಲ ಕ್ಲಿಕ್ | ಮೆನುವನ್ನು ತೆರೆಯಿರಿ (ಮೋಡ್ ಬದಲಾಯಿಸಿ / ಸೆಟ್ಟಿಂಗ್‌ಗಳು / ನಿರ್ಗಮಿಸಿ) |
| ಆಟೋ ಸ್ನ್ಯಾಪ್ | ಪರದೆ ಅಂಚುಗಳಿಂದ 20px ಒಳಗೆ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಸ್ನ್ಯಾಪ್ ಆಗುತ್ತದೆ |

### ಪ್ರದರ್ಶನ ಮೋಡ್‌ಗಳು

- **ಕಾರ್ನರ್ ವಿಜೆಟ್**: ಎಲ್ಲಾ ಮೆಟ್ರಿಕ್‌ಗಳು ಲಂಬವಾಗಿ ಸಜ್ಜುಗೊಂಡಿವೆ, ಪರದೆಯ ಯಾವುದೇ ಮೂಲೆಯಲ್ಲಿ ಇರಿಸಬಹುದು
- **ಟಾಪ್ ಬಾರ್**: ಎಲ್ಲಾ ಮೆಟ್ರಿಕ್‌ಗಳು ಅಡ್ಡಗಲಾಗಿ ಸಜ್ಜುಗೊಂಡಿವೆ, ಡೀಫಾಲ್ಟಾಗಿ ಪರದೆಯ ಮೇಲ್ಭಾಗದಲ್ಲಿ ಮಧ್ಯದಲ್ಲಿ ಇವೆ

### ಸೆಟ್ಟಿಂಗ್‌ಗಳು

- **ಪ್ರದರ್ಶನ ಮೋಡ್**: ಕಾರ್ನರ್ ವಿಜೆಟ್ / ಟಾಪ್ ಬಾರ್
- **ವಿಜೆಟ್ ಸ್ಥಾನ**: ಕೆಳಗೆ ಬಲ / ಕೆಳಗೆ ಎಡ / ಮೇಲೆ ಬಲ / ಮೇಲೆ ಎಡ
- **ವಿಂಡೋ ಪಾರದರ್ಶಕತೆ**: 30% - 100% ಒಟ್ಟು ವಿಂಡೋ ಪಾರದರ್ಶಕತೆ
- **ಹಿನ್ನೆಲೆ ಪಾರದರ್ಶಕತೆ**: 0% - 100% ಹಿನ್ನೆಲೆ ಪಾರದರ್ಶಕತೆ (ಪಠ್ಯ ಸ್ಪಷ್ಟವಾಗಿ ಇರುತ್ತದೆ)
- **ಮಾದರಿಕರಣ ಅಂತರ**: 200ms - 3000ms ಡೇಟಾ ರಿಫ್ರೆಶ್ ದರ
- **ಫಾಂಟ್ ಗಾತ್ರ**: 8 - 20 ಪಾಯಿಂಟ್‌ಗಳು
- **ಭಾಷೆ**: 20+ ಭಾಷೆಗಳು, ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಸಿಸ್ಟಮ್ ಭಾಷೆಯನ್ನು ಪತ್ತೆಮಾಡುತ್ತದೆ
- **ಪ್ರದರ್ಶನ ಮೆಟ್ರಿಕ್‌ಗಳು**: ಪ್ರತಿ ಮೆಟ್ರಿಕ್ ಅನ್ನು ಸ್ವತಂತ್ರವಾಗಿ ಆನ್/ಆಫ್ ಮಾಡಿ
- **ಪ್ರದರ್ಶನ ಸ್ವರೂಪ**: VRAM / ಮೆಮೊರಿ ಶೇಕಡಾ ಅಥವಾ ವಾಸ್ತವಿಕ ಮೌಲ್ಯಗಳ ನಡುವೆ ಬದಲಾಯಿಸಬಹುದು
- **ಆಟೋ ಸ್ಟಾರ್ಟ್**: Windows ನಲ್ಲಿ ಲಾಗಿನ್ ಮಾಡಿದಾಗ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಚಲಾಯಿಸಿ

## ಟೆಕ್ ಸ್ಟ್ಯಾಕ್

- **GUI**：Tkinter
- **ಹಾರ್ಡ್ವೇರ್ ಡೇಟಾ**：LibreHardwareMonitorLib (pythonnet ಮೂಲಕ), psutil, pynvml (NVIDIA GPU ಗಾಗಿ ಬ್ಯಾಕ್ಅಪ್)
- **ಸಿಸ್ಟಮ್ ಏಕೀಕರಣ**：pywin32 (ವಿಂಡೋ ಯಾವಾಗಲೂ ಮೇಲೆ, ಆಟೋ ಸ್ಟಾರ್ಟ್ ರೆಜಿಸ್ಟ್ರಿ)
- **ಪ್ಯಾಕೇಜಿಂಗ್**：PyInstaller

## ಪ್ರಾಜೆಕ್ಟ್ ರಚನೆ

```
ChipPeek/

├── app.ico                  # ಅಪ್ಲಿಕೇಶನ್ ಐಕಾನ್

├── requirements.txt         # Python ಅವಲಂಬನೆಗಳು
├── docs/                    # ಬಹುಭಾಷೆ ದಸ್ತಾವೇಜು

├── libs/
│   └── lhm/
│       └── lib/net472/      # LibreHardwareMonitorLib DLL
└── dist/
```

## ಸಂರಚನಾ ಫೈಲ್

`config.json` ಫೈಲ್ EXE ನ ಅದೇ ಡೈರೆಕ್ಟರಿಯಲ್ಲಿ ಉಳಿಸಲಾಗುತ್ತದೆ ಮತ್ತು ಎಲ್ಲಾ ಸರಿಹೊಂದಿಸಬಹುದಾದ ಸೆಟ್ಟಿಂಗ್‌ಗಳನ್ನು ಹೊಂದಿರುತ್ತದೆ. ಸೆಟ್ಟಿಂಗ್‌ಗಳನ್ನು ಬದಲಾಯಿಸಿದಾಗ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಉಳಿಸಲಾಗುತ್ತದೆ.
