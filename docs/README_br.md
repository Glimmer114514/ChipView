## Perzhioù

- **Diwall amzer gwir**: Frekañs CPU, gwrez CPU, frekañs GPU, gwrez GPU, arver VRAM, arver ar vemor
- **Daou mod skrammañ**: Widget korn / Barrenn uhel, cheñch gant ur c'hlik
- **Skrammañ personelaet**: Dibab frank pe vuzulioù skrammañ, trec'haoliñ etre dregantad/értaladurioù
- **Atav a-us**: Chom a-us d'an holl brenestroù, mont en-dro e c'hoarioù skramm-leun
- **Ment emgefre**: Ledander ar prenestr a azasaioù da endalc'had
- **Stumm kemmus**: Demkder ar prenestr, boullder an drekleur, ment an nodrezh kemmus holl
- **Skor yezhoù lies**: 20+ yezh, diguzhout emgefre yezh ar sistem
- **Obererezh a-feson**: Stlej kleiz evit dilec'hiañ, lañser klik deas evit arventennoù buan, emzalc'hañ ouzh riblennoù ar skramm
- **Esaou standilhonañ kemmus**: 200ms - 3000ms kemmus
- **Emloc'hañ**: Loc'hañ emgefre pa voranndigeger e Windows
- **Arver goñverzioù izel**: Roud bihan er c'hulle

## Kregiñ buan

### Arver diosern

Pellgargañ `ChipPeek.exe` ha klikkañ daouchoum evit loc'hañ (goulenn a raio emgefre aotreoù merour evit gwrez CPU ha frekañs spis).

### Loc'hañ diouzh ar vammenn

```bash
# Clone the repository
git clone https://github.com/Glimmer114514/ChipPeek.git
cd ChipPeek

# Install dependencies
pip install -r requirements.txt

# Run
python main.py
```

## Ezhommoù ar sistem

- Windows 10 / Windows 11
- Aotreoù merour (erbedet), mod-all e c'hallfe gwrez CPU ha frekañs spis chom dizimplijadus
- .NET Framework 4.7.2 pe uheloc'h (goulennet gant LibreHardwareMonitor)

## Sevel evel EXE

```bash
pip install pyinstaller
pyinstaller --noconfirm --onefile --windowed --name "ChipPeek" --icon "app.ico" --manifest "admin.manifest" --add-data "libs;libs" --add-data "app.ico;." --hidden-import "clr" --hidden-import "pynvml" --hidden-import "win32gui" --hidden-import "win32con" main.py
```

Goude ar sevel, emañ ar restr EXE e `dist/ChipPeek.exe`.

## Arver

### Obererezhioù diazez

| Obererezh | Deskrivadur |
|------|------|
| Stlej kleiz | Dilec'hiañ lec'h ar prenestr |
| Klik dehou | Digeriñ al lañser (cheñch mod / arventennoù / kuitaat) |
| Emzalc'hañ | Emzalc'hañ emgefre a-gleiz da 20px eus riblennoù ar skramm |

### Modoù skrammañ

- **Widget korn**: An holl vuzulioù en urzh a-blom, lakaet e korn forzh peseurt eus ar skramm
- **Barrenn uhel**: An holl vuzulioù en urzh a-blaen, kreizet e krec'h ar skramm dre ziouer

### Arventennoù

- **Mod skrammañ**: Widget korn / Barrenn uhel
- **Lec'h ar widget**: Traoñ dehou / Traoñ kleiz / Krec'h dehou / Krec'h kleiz
- **Demkder ar prenestr**: 30% - 100% demkder hollek ar prenestr
- **Boullder an drekleur**: 0% - 100% drekleur boullek (an testenn a chom sklaer)
- **Esaou standilhonañ**: 200ms - 3000ms feur freskadiñ ar roadennoù
- **Ment an nodrezh**: 8 - 20 poent nodrezh
- **Yezh**: 20+ yezh, diguzhout emgefre yezh ar sistem
- **Muzulioù skrammañ**: Trec'haoliñ pep muzul digenvez
- **Stumm skrammañ**: VRAM / Memor a c'hall bezañ trec'haoliet etre dregantad pe értaladurioù
- **Emloc'hañ**: Loc'hañ emgefre pa voranndigeger e Windows

## Stak teknikel

- **GUI**：Tkinter
- **Roadennoù periant**：LibreHardwareMonitorLib (dre pythonnet), psutil, pynvml (GPU NVIDIA fallback)
- **Kendeuzadur sistem**：pywin32 (prenestr a-us, enrolañ emloc'hañ)
- **Pakadur**：PyInstaller
