# ChipPeek

Windows നായി ഒരു ഇളം ഭാരമുള്ള ഫ്ലോട്ടിംഗ് ഹാർഡ്വെയർ മോനിറ്ററിംഗ് വിജെറ്റ്. CPU/GPU ഫ്രീക്വൻസി, താപനില, VRAM, മെമ്മറി ഉപയോഗം എന്നിവയുടെ റിയൽ-ടൈം മോനിറ്ററിംഗ്, ഫുൾ-സ്ക്രീൻ ആപ്പുകൾ ഉൾപ്പെടെ എല്ലായ്പ്പോഴും മുകളിൽ തുടരുന്നു.

## സവിശേഷതകൾ

- **റിയൽ-ടൈം മോനിറ്ററിംഗ്**: CPU ഫ്രീക്വൻസി, CPU താപനില, GPU ഫ്രീക്വൻസി, GPU താപനില, VRAM ഉപയോഗം, മെമ്മറി ഉപയോഗം
- **രണ്ട് ഡിസ്പ്ലേ മോഡുകൾ**: കോർണർ വിജെറ്റ് / ടോപ്പ് ബാർ, ഒരു ക്ലിക്കിൽ മാറുക
- **ഇഷ്ടാനുസൃത ഡിസ്പ്ലേ**: ഏത് മെട്രിക്കുകൾ കാണിക്കണം എന്ന് സ്വതന്ത്രമായി തിരഞ്ഞെടുക്കുക, ശതമാനം/യഥാർത്ഥ മൂല്യങ്ങൾക്കിടയിൽ മാറുക
- **എല്ലായ്പ്പോഴും മുകളിൽ**: എല്ലാ വിൻഡോകൾക്കും മുകളിൽ തുടരുന്നു, ഫുൾ-സ്ക്രീൻ ഗെയിമുകളിൽ പ്രവർത്തിക്കുന്നു
- **ഓട്ടോമാറ്റിക് വലുപ്പം**: ഉള്ളടക്കത്തിന്റെ അടിസ്ഥാനത്തിൽ വിൻഡോ വീതി സ്വയം ക്രമീകരിക്കും
- **ക്രമീകരിക്കാവുന്ന ശൈലി**: വിൻഡോ സുതാര്യത, പശ്ചാത്തല സുതാര്യത, ഫോണ്ട് വലുപ്പം - എല്ലാം ക്രമീകരിക്കാവുന്നതാണ്
- **ബഹുഭാഷാ പിന്തുണ**: 20+ ഭാഷകൾ, സ്വയം സിസ്റ്റം ഭാഷ കണ്ടെത്തും
- **എളുപ്പമുള്ള പ്രവർത്തനം**: ഇടതു ക്ലിക്ക് വലിച്ചു നീക്കുക, വലതു ക്ലിക്ക് മെനു പെട്ടെന്നുള്ള ക്രമീകരണങ്ങൾക്കായി, സ്ക്രീൻ അരികുകൾ സ്വയം സ്നാപ്പ്
- **സാമ്പിൾ ഫ്രീക്വൻസി ക്രമീകരണം**: 200ms - 3000ms ക്രമീകരിക്കാവുന്നതാണ്
- **ഓട്ടോ സ്റ്റാർട്ട്**: Windows ലെ ലോഗിനിൽ സ്വയം ആരംഭിക്കും
- **കുറഞ്ഞ വിഭവ ഉപയോഗം**: പശ്ചാത്തലിൽ കുറഞ്ഞ സ്ഥലം

## ദ്രുത ആരംഭം

### നേരിട്ടുള്ള ഉപയോഗം

`ChipPeek.exe` ഡൗൺലോഡ് ചെയ്ത് പ്രവർത്തിപ്പിക്കാൻ ഡബിൾ-ക്ലിക്ക് ചെയ്യുക (CPU താപനില, കൃത്യമായ ഫ്രീക്വൻസി വായിക്കാൻ സ്വയം അഡ്മിൻ അനുമതി അഭ്യർത്ഥിക്കും).

### സ്രോതസ്സിൽ നിന്ന് പ്രവർത്തിപ്പിക്കുക

```bash
# റിപ്പോജിറ്ററി ക്ലോൺ ചെയ്യുക
git clone https://github.com/Glimmer114514/ChipPeek.git
cd ChipPeek

# ഡിപൻഡൻസികൾ ഇൻസ്റ്റാൾ ചെയ്യുക
pip install -r requirements.txt

# പ്രവർത്തിപ്പിക്കുക
python main.py
```

## സിസ്റ്റം ആവശ്യകതകൾ

- Windows 10 / Windows 11
- അഡ്മിൻ അനുമതി (ശുപാർശിച്ചത്), അല്ലാത്തപക്ഷം CPU താപനിലയും കൃത്യമായ ഫ്രീക്വൻസിയും ലഭ്യമാകില്ല
- .NET Framework 4.7.2 അല്ലെങ്കിൽ അതിൽ മുകളിൽ (LibreHardwareMonitor ന് ആവശ്യമാണ്)

## EXE ആയി ബിൽഡ് ചെയ്യുക

```bash
pip install pyinstaller
pyinstaller --noconfirm --onefile --windowed --name "ChipPeek" --icon "app.ico" --manifest "admin.manifest" --add-data "libs;libs" --add-data "app.ico;." --hidden-import "clr" --hidden-import "pynvml" --hidden-import "win32gui" --hidden-import "win32con" main.py
```

ബിൽഡ് ചെയ്ത ശേഷം, EXE ഫയൽ `dist/ChipPeek.exe` ലാണ് ഉണ്ടായിരിക്കുന്നത്.

| വലതു ക്ലിക്ക് | മെനു തുറക്കുക (മോഡ് മാറ്റുക / ക്രമീകരണങ്ങൾ / പുറത്തുകടക്കുക) |
| ഓട്ടോ സ്നാപ്പ് | സ്ക്രീൻ അരികുകൾ നിന്നും 20px ഉള്ളിൽ സ്വയം സ്നാപ്പ് ആകും |

### ഡിസ്പ്ലേ മോഡുകൾ

- **കോർണർ വിജെറ്റ്**: എല്ലാ മെട്രിക്കുകളും ലംബമായി ക്രമീകരിച്ചിരിക്കുന്നു, സ്ക്രീനിന്റെ ഏത് മൂലയിലും സൂക്ഷിക്കാം
- **ടോപ്പ് ബാർ**: എല്ലാ മെട്രിക്കുകളും തിരശ്ചീനമായി ക്രമീകരിച്ചിരിക്കുന്നു, സ്ഥിരസ്ഥിതിയിൽ സ്ക്രീനിന്റെ മുകൾ ഭാഗത്ത് മധ്യത്തിൽ ഉണ്ട്

### ക്രമീകരണങ്ങൾ

- **ഡിസ്പ്ലേ മോഡ്**: കോർണർ വിജെറ്റ് / ടോപ്പ് ബാർ
- **വിജെറ്റ് സ്ഥാനം**: താഴെ വലത് / താഴെ ഇടത് / മുകളിൽ വലത് / മുകളിൽ ഇടത്
- **വിൻഡോ സുതാര്യത**: 30% - 100% മൊത്തം വിൻഡോ സുതാര്യത
- **പശ്ചാത്തല സുതാര്യത**: 0% - 100% പശ്ചാത്തല സുതാര്യത (ടെക്സ്റ്റ് വ്യക്തമായി തുടരും)
- **സാമ്പിൾ ഇടവേള**: 200ms - 3000ms ഡേറ്റാ റിഫ്രെഷ് നിരക്ക്
- **ഫോണ്ട് വലുപ്പം**: 8 - 20 പോയിന്റുകൾ
- **ഭാഷ**: 20+ ഭാഷകൾ, സ്വയം സിസ്റ്റം ഭാഷ കണ്ടെത്തും
- **ഡിസ്പ്ലേ മെട്രിക്കുകൾ**: ഓരോ മെട്രിക്കും സ്വതന്ത്രമായി ഓൺ/ഓഫ് ചെയ്യുക
- **ഡിസ്പ്ലേ ഫോർമാറ്റ്**: VRAM / മെമ്മറി ശതമാനം അല്ലെങ്കിൽ യഥാർത്ഥ മൂല്യങ്ങൾക്കിടയിൽ മാറ്റാം
- **ഓട്ടോ സ്റ്റാർട്ട്**: Windows ലെ ലോഗിനിൽ സ്വയം പ്രവർത്തിപ്പിക്കുക

## ടെക് സ്റ്റാക്ക്

- **GUI**：Tkinter
- **ഹാർഡ്വെയർ ഡേറ്റാ**：LibreHardwareMonitorLib (pythonnet വഴി), psutil, pynvml (NVIDIA GPU ന് ബാക്കപ്പ്)
- **സിസ്റ്റം സംയോജനം**：pywin32 (വിൻഡോ എല്ലായ്പ്പോഴും മുകളിൽ, ഓട്ടോ സ്റ്റാർട്ട് രെജിസ്ട്രി)
- **പാക്കേജിംഗ്**：PyInstaller

## കോൺഫിഗറേഷൻ ഫയൽ

`config.json` ഫയൽ EXE ന്റെ അതേ ഡയറക്ടറിയിൽ സംരക്ഷിക്കുന്നു, ക്രമീകരിക്കാവുന്ന എല്ലാ ക്രമീകരണങ്ങളും അടങ്ങിയിരിക്കുന്നു. ക്രമീകരണങ്ങൾ മാറ്റുമ്പോൾ സ്വയം സംരക്ഷിക്കും.
