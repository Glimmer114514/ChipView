## Iimpawu

- **Ukuqapha ngokwenene**: Isinqumla se-CPU, ithemprisha ye-CPU, isinqumla se-GPU, ithemprisha ye-GPU, ukusebenzisa i-VRAM, ukusebenzisa inkumbulo
- **Iindlela ezimbini zokubonisa**: I-widget yekona / Ibar ephezulu, yitshintsha ngokucofa kakunye
- **Ukubonisa okulungiselelweyo**: Khetha ngokukhululekileyo ukuba zeziphi na iindidi eziza kuboniswa, tshintsha phakathi kwamaphesenti/amaxabiso angekhoyo
- **Ukuhlala phezulu kusoloko**: Uhlala phezulu kuzo zonke iifestile, usebenza kwiimidlalo yesikrini esizeleyo
- **Ukulinganisa ngokuzenzakalela**: Ububanzi befestile buqwalasela ngokuzenzakalela ukulingana nomxholo
- **Isimbo esilungiselelweyo**: Ukubonakala kwefestile, ukubonakala kwasemva, ubungakanani befonti zonke ziyalungiselelwa
- **Inkxaso yoolwimi oluninzi**: Iilwimi ezingaphezu kwe-20, ifumana ngokuzenzakalela ulwimi lwesixokelelwano
- **Ukuhamba kalula**: Tsala ngasekhohlo ukuze uhambise, cofa ngasekunze ikhusi elikhawulezayo, namathisela ngokuzenzakalela kumda wesikrini
- **Isampuli esilungiselelweyo**: 200ms - 3000ms iyalungiselelwa
- **Qala ngokuzenzakalela**: Qala ngokuzenzakalela xa ungena kwi-Windows
- **Ukusetyenziswa kwezixhobo ezincinci**: Isithuba esincinci ngasemva

## Qala Ngokukhawuleza

### Ukusetyenziswa Ngqo

Layisha `ChipPeek.exe` kwaye ucofe kabini ukuqalisa (iza kucela ilungelo lomlawuli ukuze ifunde ukubanda kwe-CPU kunye nesinqumla esichanekileyo).

### Qalisakusuka Kumthombo

```bash
# Kopisha isilabo
git clone https://github.com/Glimmer114514/ChipPeek.git
cd ChipPeek

# Faka izixhobo ezixhomekeekayo
pip install -r requirements.txt

# Qalisa
python main.py
```

## Iimfuno ZeNkqubo

- Windows 10 / Windows 11
- Ilungelo lomlawuli (kuyacetyiswa), kungenjalo ukubanda kwe-CPU kunye nesinqumla esichanekileyo kungafumaneki
- .NET Framework 4.7.2 okanye ngaphezulu (LibreHardwareMonitor iyayifuna)

## Yakha Njenge-EXE

```bash
pip install pyinstaller
pyinstaller --noconfirm --onefile --windowed --name "ChipPeek" --icon "app.ico" --manifest "admin.manifest" --add-data "libs;libs" --add-data "app.ico;." --hidden-import "clr" --hidden-import "pynvml" --hidden-import "win32gui" --hidden-import "win32con" main.py
```

Emva kokuyakha, ifayile ye-EXE itholakala kwi `dist/ChipPeek.exe`.

## Indlela Yokusetyenziswa

### Iintshukumo Eziyimpilo

| Intshukumo | Inkcazelo |
|------|------|
| Tsala ngasekhohlo | Hambisa indawo yefestile |
| Cofa ngasekunene | Vula ikhusi (tshintsha indlela / izicwangciso / phuma) |
| Namathisela ngokuzenzakalela | Izibambe ngokuzenzakalela ngaphakathi kwe-20px yomda wesikrini |

### Iindlela Zokubonisa

- **I-widget yekona**: Zonke iindidi ziqulunqwe ngobude, zingabekwa nakuphi na ikona yomda wesikrini
- **Ibar ephezulu**: Zonke iindidi ziqulunqwe ngobubanzi, zisembindini phezulu kwesikrini ngokumiselweyo

### Izicwangciso

- **Indlela yokubonisa**: I-widget yekona / Ibar ephezulu
- **Indawo ye-widget**: Ezantsi ngasekunene / Ezantsi ngasekhohlo / Phezulu ngasekunene / Phezulu ngasekhohlo
- **Ukubonakala kwefestile**: 30% - 100% ukubonakala kwefestile jikelele
- **Ukubonakala kwasemva**: 0% - 100% imvelaphi ebonakalayo (umbhalo uhlala ucace)
- **Ithuba lesampulu**: 200ms - 3000ms isantya sokuvuselela idatha
- **Ubungakanani befonti**: 8 - 20 iponti fonti
- **Ulwimi**: Iilwimi ezingaphezu kwe-20, ifumana ngokuzenzakalela ulwimi lwesixokelelwano
- **Iindidi ezibonisiweyo**: Tshintsha ngeyodwa nganye ngokuzodwa
- **Ifomathi yokubonisa**: I-VRAM / Inkumbulo ingatshintshwa phakathi kwamaphesenti okanye amaxabiso angekhoyo
- **Qala ngokuzenzakalela**: Qala ngokuzenzakalela xa ungena kwi-Windows

## Inqanaba Lobuchwepheshe

- **GUI**：Tkinter
- **Idata yezixhobo**：LibreHardwareMonitorLib (nge-pythonnet), psutil, pynvml (indawo yokubuyela emva kwe-NVIDIA GPU)
- **Ukuhlanganiswa kwesixokelelwano**：pywin32 (ifestile ephezulu, irejista yoqalo ngokuzenzakalela)
- **Ukupakisha**：PyInstaller
