# ChipPeek

Windows માટે એક હળવું વજનનું ફ્લોટિંગ હાર્ડવેર મોનિટરિંગ વિજેટ. CPU/GPU ફ્રીક્વન્સી, તાપમાન, VRAM અને મેમરી વપરાશનું રીઅલ-ટાઇમ મોનિટરિંગ, ફુલ-સ્ક્રીન એપ્સ સહિત હંમેશા ટોચ પર રહે છે.

## લક્ષણો

- **રીઅલ-ટાઇમ મોનિટરિંગ**: CPU ફ્રીક્વન્સી, CPU તાપમાન, GPU ફ્રીક્વન્સી, GPU તાપમાન, VRAM વપરાશ, મેમરી વપરાશ
- **બે ડિસ્પ્લે મોડ**: કોર્નર વિજેટ / ટોપ બાર, એક ક્લિકથી સ્વિચ કરો
- **કસ્ટમાઇઝ્ડ ડિસ્પ્લે**: કઈ મેટ્રિક્સ બતાવવી તે મુક્તપણે પસંદ કરો, ટકાવારી/વાસ્તવિક મૂલ્યો વચ્ચે સ્વિચ કરો
- **હંમેશા ટોચ પર**: બધી વિન્ડોની ટોચ પર રહે છે, ફુલ-સ્ક્રીન ગેમ્સમાં કામ કરે છે
- **સ્વયંસંચાલિત કદ**: સામગ્રીના આધારે વિન્ડો પહોળાઈ આપોઆપ સમાયોજિત થાય છે
- **સમાયોજ્ય શૈલી**: વિન્ડો પારદર્શિતા, પૃષ્ઠભૂમિ પારદર્શિતા, ફોન્ટ કદ - બધું સમાયોજ્ય
- **બહુભાષી સમર્થન**: 20+ ભાષાઓ, આપોઆપ સિસ્ટમ ભાષા શોધે છે
- **સરળ ઓપરેશન**: ડાબા ક્લિકથી ડ્રેગ કરો અને ખસેડો, જમણા ક્લિક મેનુ ઝડપી સેટિંગ્સ માટે, સ્ક્રીન કિનારે સ્વયંસંચાલિત સ્નેપ
- **સેમ્પલિંગ ફ્રીક્વન્સી સમાયોજન**: 200ms - 3000ms સમાયોજ્ય
- **ઓટો સ્ટાર્ટ**: Windows માં લૉગિન પર આપોઆપ શરૂ થાય છે
- **ઓછું સંસાધન વપરાશ**: પૃષ્ઠભૂમિમાં ન્યૂનતમ જગ્યા

## ઝડપી શરૂઆત

### સીધો ઉપયોગ

`ChipPeek.exe` ડાઉનલોડ કરો અને ચલાવવા માટે ડબલ-ક્લિક કરો (CPU તાપમાન અને ચોક્કસ ફ્રીક્વન્સી વાંચવા માટે આપોઆપ વહીવટી પરવાનગી માંગશે).

### સ્ત્રોતમાંથી ચલાવો

```bash
# રિપોઝિટરી ક્લોન કરો
git clone https://github.com/Glimmer114514/ChipPeek.git
cd ChipPeek

# ડિપેન્ડન્સી ઇન્સ્ટોલ કરો
pip install -r requirements.txt

# ચલાવો
python main.py
```

## સિસ્ટમ આવશ્યકતાઓ

- Windows 10 / Windows 11
- વહીવટી પરવાનગી (ભલામણ કરવામાં આવે છે), અન્યથા CPU તાપમાન અને ચોક્કસ ફ્રીક્વન્સી ઉપલબ્ધ ન હોઈ શકે
- .NET Framework 4.7.2 અથવા ઉચ્ચ (LibreHardwareMonitor માટે જરૂરી)

## EXE તરીકે બિલ્ડ કરો

```bash
pip install pyinstaller
pyinstaller --noconfirm --onefile --windowed --name "ChipPeek" --icon "app.ico" --manifest "admin.manifest" --add-data "libs;libs" --add-data "app.ico;." --hidden-import "clr" --hidden-import "pynvml" --hidden-import "win32gui" --hidden-import "win32con" main.py
```

બિલ્ડ કર્યા પછી, EXE ફાઇલ `dist/ChipPeek.exe` માં છે.

## ઉપયોગ

### મૂળભૂત ઓપરેશન્સ

| ઓપરેશન | વિવરણ |
|------|------|
| ડાબો ક્લિક ડ્રેગિંગ | વિન્ડો સ્થિતિ ખસેડો |
| જમણો ક્લિક | મેનુ ખોલો (મોડ સ્વિચ કરો / સેટિંગ્સ / બહાર નીકળો) |
| ઓટો સ્નેપ | સ્ક્રીન કિનારથી 20px અંદર આપોઆપ સ્નેપ થાય છે |

### ડિસ્પ્લે મોડ્સ

- **કોર્નર વિજેટ**: બધા મેટ્રિક્સ ઊભી ગોઠવણમાં ગોઠવાયેલા છે, સ્ક્રીનના કોઈપણ ખૂણામાં રાખી શકાય છે
- **ટોપ બાર**: બધા મેટ્રિક્સ આડી ગોઠવણમાં ગોઠવાયેલા છે, ડિફૉલ્ટ રૂપે સ્ક્રીનની ટોચ પર મધ્યમાં છે

### સેટિંગ્સ

- **ડિસ્પ્લે મોડ**: કોર્નર વિજેટ / ટોપ બાર
- **વિજેટ સ્થિતિ**: નીચે જમણી / નીચે ડાબી / ટોચ જમણી / ટોચ ડાબી
- **વિન્ડો પારદર્શિતા**: 30% - 100% કુલ વિન્ડો પારદર્શિતા
- **પૃષ્ઠભૂમિ પારદર્શિતા**: 0% - 100% પૃષ્ઠભૂમિ પારદર્શિતા (ટેક્સ્ટ સ્પષ્ટ રહે છે)
- **સેમ્પલિંગ અંતરાલ**: 200ms - 3000ms ડેટા રિફ્રેશ દર
- **ફોન્ટ કદ**: 8 - 20 પોઈન્ટ
- **ભાષા**: 20+ ભાષાઓ, આપોઆપ સિસ્ટમ ભાષા શોધે છે
- **ડિસ્પ્લે મેટ્રિક્સ**: દરેક મેટ્રિક્સ સ્વતંત્ર રીતે ઓન/ઓફ કરો
- **ડિસ્પ્લે ફોર્મેટ**: VRAM / મેમરી ટકાવારી અથવા વાસ્તવિક મૂલ્યો વચ્ચે સ્વિચ કરી શકાય છે
- **ઓટો સ્ટાર્ટ**: Windows માં લૉગિન પર આપોઆપ ચલાવો

## ટેક સ્ટેક

- **GUI**：Tkinter
- **હાર્ડવેર ડેટા**：LibreHardwareMonitorLib (pythonnet દ્વારા), psutil, pynvml (NVIDIA GPU માટે બેકઅપ)
- **સિસ્ટમ ઈન્ટિગ્રેશન**：pywin32 (વિન્ડો હંમેશા ટોચ પર, ઓટો સ્ટાર્ટ રજિસ્ટ્રી)
- **પેકેજિંગ**：PyInstaller

```
ChipPeek/

├── app.png                  # આઇકન પૂર્વાવલોકન

├── docs/                    # બહુભાષી દસ્તાવેજીકરણ
```

## કન્ફિગરેશન ફાઇલ

`config.json` ફાઇલ EXE ની જેમ જ ડિરેક્ટરીમાં સેવ થાય છે અને તમામ સમાયોજ્ય સેટિંગ્સ ધરાવે છે. સેટિંગ્સ બદલાય ત્યારે આપોઆપ સેવ થાય છે.
