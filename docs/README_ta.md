# ChipPeek

Windows க்கான ஒரு இலகுரக மிதக்கும் கணினி மானிட்டர் சாதனம். CPU/GPU அதிர்வெண், வெப்பநிலை, VRAM மற்றும் நினைவகப் பயன்பாட்டின் உண்மை நேர மானிட்டரிங், முழு திரை பயன்பாடுகளுடன் எப்போதும் மேலே இருக்கும்.

## அம்சங்கள்

- **உண்மை நேர மானிட்டரிங்**: CPU அதிர்வெண், CPU வெப்பநிலை, GPU அதிர்வெண், GPU வெப்பநிலை, VRAMபயன்பாடு, நினைவகப் பயன்பாடு
- **இரண்டு காட்சி பயன்முறைகள்**: மூலை விட்ஜெட் / டாப் பார், ஒரு கிளிக்கில் மாற்றவும்
- **தனிப்பயன் காட்சி**: எந்த குறிப்பான்களைக் காட்ட வேண்டும் என்பதை சுதந்திரமாகத் தேர்ந்தெடுக்கவும், சதவீதம்/உண்மை மதிப்புகளுக்கு இடையே மாற்றவும்
- **எப்போதும் மேலே**: அனைத்து சாளரங்களுக்கும் மேலே இருக்கும், முழு திரை விளையாட்டுகளில் வேலை செய்கிறது
- **தானியங்கி அளவு**: உள்ளடக்கத்தின் அடிப்படையில் சாளரத்தின் அகலம் தானாக சரிசெய்யப்படும்
- **சரிசெய்யக்கூடிய பாணி**: சாளர வெளிப்படுத்து, பின்னணி வெளிப்படுத்து, எழுத்துரு அளவு - அனைத்தும் சரிசெய்யக்கூடியவை
- **பல மொழி ஆதரவு**: 20+ மொழிகள், தானாக கணினி மொழியைக் கண்டறியும்
- **எளிய செயல்பாடு**: இடது கிளிக் இழுத்து நகர்த்தவும், வலது கிளிக் மெனு விரைவான அமைப்புகளுக்கும், திரை விளிம்புகளில் தானியங்கி பிடிப்பு
- **தரவுத் தொகுப்பு அதிர்வெண் சரிவு**: 200ms - 3000ms சரிசெய்யக்கூடியது
- **தானியங்கி தொடக்கம்**: Windows இல் உள்நுழையும் போது தானாக தொடங்கும்
- **குறைந்த வள பயன்பாடு**: பின்னணியில் குறைந்தபட்ச இடம்

## விரைவான தொடக்கம்

### நேரடி பயன்பாடு

`ChipPeek.exe` ஐ பதிவிறக்கி இயக்க இருமுறை கிளிக் செய்யவும் (CPU வெப்பநிலை மற்றும் துல்லியமான அதிர்வெண்ணைப் படிக்க தானாக நிர்வாக அனுமதியைக் கோரும்).

### மூலத்திலிருந்து இயக்கவும்

```bash
# களஞ்சியத்தை குளோன் செய்யவும்
git clone https://github.com/Glimmer114514/ChipPeek.git
cd ChipPeek

# சார்புகளை நிறுவவும்
pip install -r requirements.txt

# இயக்கவும்
python main.py
```

## கணினி தேவைகள்

- Windows 10 / Windows 11
- நிர்வாக அனுமதி (பரிந்துரைக்கப்படுகிறது), இல்லையெனில் CPU வெப்பநிலை மற்றும் துல்லியமான அதிர்வெண் கிடைக்காது
- .NET Framework 4.7.2 அல்லது அதற்கு மேல் (LibreHardwareMonitor தேவை)

## EXE ஆக உருவாக்கவும்

```bash
pip install pyinstaller
pyinstaller --noconfirm --onefile --windowed --name "ChipPeek" --icon "app.ico" --manifest "admin.manifest" --add-data "libs;libs" --add-data "app.ico;." --hidden-import "clr" --hidden-import "pynvml" --hidden-import "win32gui" --hidden-import "win32con" main.py
```

உருவாக்கிய பிறகு, EXE கோப்பு `dist/ChipPeek.exe` இல் உள்ளது.

| செயல்பாடு | விவரம் |

### காட்சி பயன்முறைகள்

- **மூலை விட்ஜெட்**: அனைத்து குறிப்பான்களும் செங்குத்தாக அமைக்கப்பட்டுள்ளன, திரையின் ஏதாவது ஒரு மூலையில் வைக்கலாம்
- **டாப் பார்**: அனைத்து குறிப்பான்களும் கிடைமட்டமாக அமைக்கப்பட்டுள்ளன, இயல்புநிலையில் திரையின் மேல் பாகத்தில் நடுவில் உள்ளன

### அமைப்புகள்

- **காட்சி பயன்முறை**: மூலை விட்ஜெட் / டாப் பார்
- **விட்ஜெட் நிலை**: கீழே வலது / கீழே இடது / மேலே வலது / மேலே இடது
- **சாளர வெளிப்படுத்து**: 30% - 100% மொத்த சாளர வெளிப்படுத்து
- **பின்னணி வெளிப்படுத்து**: 0% - 100% பின்னணி வெளிப்படுத்து (உரை தெளிவாக இருக்கும்)
- **மாதிரி இடைவெளி**: 200ms - 3000ms தரவு புதுப்பிப்பு விகிதம்
- **எழுத்துரு அளவு**: 8 - 20 பாயின்ட்கள்
- **மொழி**: 20+ மொழிகள், தானாக கணினி மொழியைக் கண்டறியும்
- **காட்சி குறிப்பான்கள்**: ஒவ்வொரு குறிப்பானையும் சுயாதீனமாக இயக்க/நிறுத்தவும்
- **காட்சி வடிவம்**: VRAM / நினைவகம் சதவீதம் அல்லது உண்மை மதிப்புகளுக்கு இடையே மாற்றலாம்
- **தானியங்கி தொடக்கம்**: Windows இல் உள்நுழையும் போது தானாக இயக்கவும்

## தொழில்நுட்ப அமைப்பு

- **GUI**：Tkinter
- **வன்பொருள் தரவு**：LibreHardwareMonitorLib (pythonnet மூலம்), psutil, pynvml (NVIDIA GPU க்கு காப்பு)
- **கணினி ஒருங்கிணைப்பு**：pywin32 (சாளரம் எப்போதும் மேலே, தானியங்கி தொடக்கம் பதிவு)
- **தொகுப்பு**：PyInstaller

```
ChipPeek/

├── config.py                # கட்டமைப்பு நிர்வாகம்
├── i18n.py                  # சர்வதேசமயமாக்கல்
├── generate_icon.py         # ஐகான் உருவாக்கும் ஸ்கிரிப்ட்
├── admin.manifest           # UAC நிர்வாக அனுமதி மேனிஃபெஸ்ட்
├── app.ico                  # பயன்பாட்டு ஐகான்
├── app.png                  # ஐகான் முன்னோட்டம்
├── requirements.txt         # Python சார்புகள்
├── docs/                    # பல மொழி ஆவணங்கள்
│   └── README_*.md
├── i18n/                    # மொழிபெயர்ப்பு கோப்புகள்
│   └── *.json
├── libs/
│   └── lhm/
│       └── lib/net472/      # LibreHardwareMonitorLib DLL
└── dist/
    └── ChipPeek.exe         # கட்டமைக்கப்பட்ட செயல்கூறு
```

## கட்டமைப்பு கோப்பு

`config.json` கோப்பு EXE உடன் அதே கோப்புறையில் சேமிக்கப்படுகிறது மற்றும் அனைத்து சரிசெய்யக்கூடிய அமைப்புகளையும் கொண்டுள்ளது. அமைப்புகள் மாற்றும் போது தானாக சேமிக்கப்படும்.
