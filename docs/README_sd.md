## خاصيتون

- **حقيقي وقت نگراني**: CPU فريڪوينسي، CPU گرمي پد، GPU فريڪوينسي، GPU گرمي پد، VRAM استعمال، ميموري استعمال
- **ٻه ڊسپلي موڊ**: ڪنڊ وجاٽ / مٿيون بار، هڪ ڪلڪ تي تبديل
- **ڊسپلي جي ترتيب**: ڪهڙا ماپ ڏيکارڻا آهن انهن جي آزاد چونڊ، سيڪڙو/اصلي قدرن جي وچ ۾ تبديلي
- **هميشه مٿي**: سڀني ونڊوز مٿان رهي ٿو، فل اسڪرين ران ۾ ڪم ڪري ٿو
- **خودڪار ماپ**: ونڊو ويڪر مواد جي مطابق خودڪار ترتيب
- **ترتيب ڏيڻ جوڳو انداز**: ونڊو شفافيت، پس منظر شفافيت، فونٽ ماپ سڀ ترتيب ڏيڻ جوڳا
- **گهڻ ٻولي سپورت**: 20+ ٻوليون، سسٽم ٻولي خودڪار سڃاڻي
- **آسان هلائڻ**: کاٻو ڇڪي حرکت، ساڄي ڪلڪ مينو لاءِ چڱائي سيٽنگون، اسڪرين ڪنڊن تي خودڪار چنبڻ
- **نموني جي ترتيب**: 200ms - 3000ms ترتيب ڏيڻ جوڳو
- **خودڪار شروعات**: Windows لاگ ان تي خودڪار شروعات
- **گهٽ وسائل استعمال**: پس منظر ۾ گهٽ وسائل استعمال

## چڱائي شروعات

### سڌو استعمال

`ChipPeek.exe` ڊائون لوڊ ڪريو ۽ ٻه ڀيرا ڪلڪ ڪريو هلائڻ لاءِ (CPU گرمي پد ۽ صحيح فريڪوينسي پڙهڻ لاءِ خودڪار طور ايڊمن حق گهرندو).

### سورس مان هلائڻ

```bash
# ريپوزٽري ڪلون ڪريو
git clone https://github.com/Glimmer114514/ChipPeek.git
cd ChipPeek

# انحصار نصب ڪريو
pip install -r requirements.txt

# هلائڻ
python main.py
```

## سسٽم گهرجون

- Windows 10 / Windows 11
- ايڊمن حق (سفارش ٿيل)، نه ته CPU گرمي پد ۽ صحيح فريڪوينسي نہ ملڻ جوڳي آهي
- .NET Framework 4.7.2 يا مٿي (LibreHardwareMonitor لاءِ گهرجي)

## EXE طور ٺاهڻ

```bash
pip install pyinstaller
pyinstaller --noconfirm --onefile --windowed --name "ChipPeek" --icon "app.ico" --manifest "admin.manifest" --add-data "libs;libs" --add-data "app.ico;." --hidden-import "clr" --hidden-import "pynvml" --hidden-import "win32gui" --hidden-import "win32con" main.py
```

ٺاهڻ کان پوءِ، EXE فائيل `dist/ChipPeek.exe` ۾ آهي.

## استعمال

### بنيادي عمل

| عمل | تشريح |
|------|------|
| کاٻو ڇڪي | ونڊو پوزيشن منتقل ڪريو |
| ساڄو ڪلڪ | مينيو کوليو (موڊ تبديل / سيٽنگون / بند) |
| خودڪار چنبڻ | اسڪرين ڪنڊن جي 20px اندر خودڪار چنبڻ |

### ڊسپلي موڊ

- **ڪنڊ وجاٽ**: سڀ ماپون عمودي ترتيب سان، اسڪرين جي ڪنهن به ڪنڊ ۾ رکي سگهجن ٿيون
- **مٿيون بار**: سڀ ماپون افقي ترتيب سان، ڊفالٽ طور اسڪرين جي مٿئين حصي ۾ وچ ۾

### سيٽنگون

- **ڊسپلي موڊ**: ڪنڊ وجاٽ / مٿيون بار
- **وجاٽ پوزيشن**: هيٺڙو ساڄو / هيٺڙو کاٻو / مٿي ساڄو / مٿي کاٻو
- **ونڊو شفافيت**: 30% - 100% مجموعي ونڊو شفافيت
- **پس منظر شفافيت**: 0% - 100% پس منظر شفاف (ٽيڪسٽ واضح رهي ٿو)
- **نموني جو وقفو**: 200ms - 3000ms ڊيٽا تازه ڪاري شرح
- **فونٽ ماپ**: 8 - 20 پوائنٽ فونٽ
- **ٻولي**: 20+ ٻوليون، سسٽم ٻولي خودڪار سڃاڻي
- **ڊسپلي ماپون**: هر ماپ کي الڳ طور آن/آف ڪري سگهبو
- **ڊسپلي فارميٽ**: VRAM / ميموري سيڪڙو يا اصلي قدرن ۾ تبديل ڪري سگهجن ٿيون
- **خودڪار شروعات**: Windows ۾ لاگ ان تي خودڪار هلائڻ

## ٽيڪنيڪل اسٽيڪ

- **GUI**：Tkinter
- **هارڊويئر ڊيٽا**：LibreHardwareMonitorLib (pythonnet ذريعي)، psutil، pynvml (NVIDIA GPU متبادل)
- **سسٽم انٽيگريشن**：pywin32 (ونڊو مٿي، خودڪار شروعات رجسٽري)
- **پئڪيجنگ**：PyInstaller

## پروجيڪٽ جوڙجڪ

```
ChipPeek/
├── main.py                  # داخلا نقطو
├── monitor_window.py        # ونڊو UI ۽ تعامل منطق
├── hardware_monitor.py      # هارڊويئر ڊيٽا گڏ ڪرڻ
├── config.py                # سيٽنگون جو انتظام
├── i18n.py                  # بين الاقواميڪرن
├── generate_icon.py         # آئيڪن ٺاهڻ اسڪرپٽ
├── admin.manifest           # UAC ايڊمن حق مانيفيسٽ
├── app.ico                  # ايپليڪيشن آئيڪن
├── app.png                  # آئيڪن پيش نظارو
├── requirements.txt         # Python انحصار
├── docs/                    # گهڻ ٻولي دستاويز
│   └── README_*.md
├── i18n/                    # ترجمو فائيلون
│   └── *.json
├── libs/
│   └── lhm/

└── dist/
    └── ChipPeek.exe         # مرتب ٿيل ايگزيڪيوٽبل
```

## سيٽنگون فائيل

`config.json` فائيل EXE سان گڏ ساڳئي ڊاريڪٽري ۾ محفوظ ٿيل آهي، سڀ ترتيب ڏيڻ جوگيون سيٽنگون تي مشتمل. سيٽنگون تبديل ٿيڻ تي خودڪار محفوظ ٿين ٿيون.
