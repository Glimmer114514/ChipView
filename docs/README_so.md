## Astayaasha

- **Kormeerka goonta ah**: Tirtirka CPU, kuleylka CPU, tirtirka GPU, kuleylka GPU, isticmaalka VRAM, isticmaalka xusuusta
- **Hababka muuqaalka laba**: Widget da xagga / Bar sare, beddel hal taabasho
- **Muuqaal la beddeli karo**: Xulo xor ah walxaha la muuqdo, beddel dhexdhexaad/qiyaasaha dhabta ah
- **Had iyo jeer kor**: Wuu kor fadhiyaa daaqadaha oo dhan, wuxuu shaqeeyaa ciyaaraha fullscreen
- **Cabbirka isbeddelka**: Balaca daaqada wuu isbeddelaa si uu u qabto waxa ku jira
- **Qaab la beddeli karo**: Cadaaladka daaqada, cadadka asalka, cabbirka qoraalka dhammaantood waa la beddeli karaa
- **Taageerada luqadaha badan**: Luqado 20+, ogaanshaha luqadda nidaamka si toos ah
- **Hawl fudud**: Riix bidix dhaqaaq daaqada, riix midig menu oo degdeg ah, qabsasho toos ah daaha shaashadda
- **Dhibicaha sampling la beddeli karo**: 200ms - 3000ms la beddeli karo
- **Bilow si toos ah**: Bilow si toos ah markaad gasho Windows
- **Isticmaalka khayraadka hoose**: Qaybtan dambe isticmaal badan yahay

## Bilow Degdeg

### Isticmaal Si Toos Ah

Soo deji `ChipPeek.exe` oo laba jeer taabo si aad u bilowdo (wuxuu si toos ah u codsan doonaa xujooyinka admin ee kuleylka CPU iyo akhrinta tirtirka saxda ah).

### U shaqeey Isaga soo baxay Source

```bash
# Soo qaad repository
git clone https://github.com/Glimmer114514/ChipPeek.git
cd ChipPeek

# Soo deji dependencies
pip install -r requirements.txt

# Shaqeey
python main.py
```

## Shuruudaha Nidaamka

- Windows 10 / Windows 11
- Xujooyinka admin (waa la gudboon yahay), haddii kale kuleylka CPU iyo tirtirka saxda ah lama heli karo
- .NET Framework 4.7.2 ama ka sarreysa (waa loo baahan yahay LibreHardwareMonitor)

## Dhis EXE

```bash
pip install pyinstaller
pyinstaller --noconfirm --onefile --windowed --name "ChipPeek" --icon "app.ico" --manifest "admin.manifest" --add-data "libs;libs" --add-data "app.ico;." --hidden-import "clr" --hidden-import "pynvml" --hidden-import "win32gui" --hidden-import "win32con" main.py
```

Marka la dhammeeyo dhismaha, faylka EXE wuxuu ku yaalaa `dist/ChipPeek.exe`.

## Isticmaalka

### Hawlaha Aasaaska

| Hawlasha | Sharaxaad |
|------|------|
| Riix bidix | Dhaqaaq booska daaqada |
| Riix midig | Fur menu (beddel habka / dejinta / ka bax) |
| Qabsasho toos ah | Si toos ah ayuu u qabsanayaa gudaha 20px ee daaha shaashadda |

### Hababka Muuqaalka

- **Widget da xagga**: Dhammaan tilmaamayaasha ayaa kor u kala yaalaa, waxaana la dhigi karaa xag kasta oo shaashadda ah
- **Bar sare**: Dhammaan tilmaamayaasha ayaa jiifan, si aasaasi ah wuxuu ku yaalaa bartamaha shaashadda sare

### Dejinta

- **Habka muuqaalka**: Widget da xagga / Bar sare
- **Booska widget**: Hoose midig / Hoose bidix / Sare midig / Sare bidix
- **Cadaaladka daaqada**: 30% - 100% cadaaladka daaqada oo dhan
- **Cadadka asalka**: 0% - 100% asalka oo cad (qoraalku wuu cadnaadaa)
- **Dhibicaha sampling**: 200ms - 3000ms xawaaraha cusboonaysiinta xogta
- **Cabbirka qoraalka**: 8 - 20 dhibco qoraal ah
- **Luqadda**: Luqado 20+, ogaanshaha luqadda nidaamka si toos ah
- **Tilmaamayaasha la muuqdo**: Fur/xidb tilmaan kasta si gaar ah
- **Habka muuqaalka**: VRAM / Xusuus waxaa la beddeli karaa boqolkiya ama qiimaha dhabta ah
- **Bilow si toos ah**: Si toos ah ayuu u bilaabmayaa markaad gasho Windows

## Stack Teknolojiyada

- **GUI**：Tkinter
- **Xogta hardware**：LibreHardwareMonitorLib (iyada oo loo marayo pythonnet), psutil, pynvml (NVIDIA GPU fallback)
- **Isku xirka nidaamka**：pywin32 (daaqada kor, bilowga toos ah ee registry)
- **Udhisidda**：PyInstaller
