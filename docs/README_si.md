## ලක්ෂණ

- **යථාර්ථවිරූප මොනිටර් කිරීම**: CPU සංඛ්‍යාතය, CPU උෂ්ණත්වය, GPU සංඛ්‍යාතය, GPU උෂ්ණත්වය, VRAM භාවිතය, මතක භාවිතය
- **ද්විත්ව ප්‍රදර්ශන ආකාර**: කෙළපෙළ විජට් / ඉහළ තීරුව, එක් ක්ලික්යකින් මාරු වීම
- **අභිරුචි කළ හැකි ප්‍රදර්ශනය**: පෙන්වීමට අවශ්‍ය මානනීය අංග නිදහසේ තෝරා ගන්න, ප්‍රතිශතය/සැබෑ අගය මාරු කරන්න
- **සැමවිටම ඉහළින්**: සියලුම ජනෙල්වලින් ඉහළින් පවතී, පූර්ණ තිර ක්‍රීඩා වලදී ක්‍රියා කරයි
- **ස්වයංක්‍රීය ප්‍රමාණය සකස් කිරීම**: ජනෙල පළල අන්තර්ගතයට ගැලවීම ස්වයංක්‍රීයව සකස් වේ
- **සකස් කළ හැකි ශෛලිය**: ජනෙල විනාශකාරිතාව, පසුබිම විනාශකාරිතාව, අකුර ප්‍රමාණය සියල්ලම සකස් කළ හැක
- **බහු-භාෂා සහාය**: 20+ භාෂා, පද්ධති භාෂාව ස්වයංක්‍රීයව හඳුනාගනී
- **පහසු මෙහෙයුම**: වම් ඇදගෙන යෑමෙන් ගෙනයන්න, දකුණු ක්ලික් මෙනුවෙන් ක්ෂණික සැකසුම්, වේදයෙ දිගට ස්වයංක්‍රීයව අමුණන්න
- **වින්යාස කළ හැකි නියැදීම**: 200ms - 3000ms සකස් කළ හැක
- **ස්වයංක්‍රීය ආරම්භය**: Windows ප්‍රවේශයේදී ස්වයංක්‍රීයව ආරම්භ වන්න
- **අඩු සම්පත් භාවිතය**: පසුබිමේ අවම ප්‍රමාණයේ භාවිතය

## ක්ෂණික ආරම්භය

### සෘජු භාවිතය

`ChipPeek.exe` බාගත කර ඩබල් ක්ලික් කිරීමෙන් ධාවනය කරන්න (CPU උෂ්ණත්වය සහ නිවැරදි සංඛ්‍යාතය කියවීම සඳහා ස්වයංක්‍රීයව පරිපාලන අයිතිය ඉල්ලා සිටේ).

### මූලාශ්‍රයෙන් ධාවනය කරන්න

```bash
# රිපෝසිටරය ක්ලෝන කරන්න
git clone https://github.com/Glimmer114514/ChipPeek.git
cd ChipPeek

# ආශ්‍රිත ස්ථාපනය කරන්න
pip install -r requirements.txt

# ධාවනය කරන්න
python main.py
```

## තාක්ෂණික කූඩුව

- **GUI**：Tkinter
- **දෘඩාංග දත්ත**：LibreHardwareMonitorLib (pythonnet හරහා), psutil, pynvml (NVIDIA GPU යටත් විජිතය)
- **පද්ධති ඒකාබද්ධතාවය**：pywin32 (ජනෙල ඉහළම, ස්වයංක්‍රීය-ආරම්භය රෙජිස්ට්‍රි)
- **පැකේජය**：PyInstaller

## ව්‍යාපෘති ව්‍යුහය

```
ChipPeek/
├── main.py                  # ප්‍රවේශ ස්ථානය
├── monitor_window.py        # ජනෙල UI සහ අන්තර්ක්‍රියා තර්කය
├── hardware_monitor.py      # දෘඩාංග දත්ත එකතු කිරීම
├── config.py                # වින්‍යාස කළමනාකරණය
├── i18n.py                  # ජාත්‍යන්තරීකරණය
├── generate_icon.py         # නිරූපක ජනන ස්ක්‍රිප්ට්
├── admin.manifest           # UAC පරිපාලන අයිති ප්‍රකාශනය
├── app.ico                  # යෙදුමේ නිරූපකය
├── app.png                  # නිරූපක පෙරදසුන
├── requirements.txt         # Python ආශ්‍රිත
├── docs/                    # බහු-භාෂා ලේඛන
│   └── README_*.md
├── i18n/                    # පරිවර්තන ගොනු
│   └── *.json
├── libs/
│   └── lhm/
│       └── lib/net472/      # LibreHardwareMonitorLib DLL
└── dist/
    └── ChipPeek.exe         # සම්පාදිත ක්‍රියාත්මක ගොනුව
```

## වින්‍යාස ගොනුව

`config.json` ගොනුව EXE හා එකම ඩිරෙක්ටරියේ සුරැකි ඇත, සියලුම සකස් කළ හැකි සැකසුම් අඩංගු වේ. සැකසුම් සංශෝධිත විට ස්වයංක්‍රීයව සුරැකේ.
