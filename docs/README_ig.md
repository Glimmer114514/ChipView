## Ngwa Ọrụ

- **Nyocha n'oge ebie**: Mitar CPU, ọnụ CPU, mitar GPU, ọnụ GPU, ojiji VRAM, ojiji echiche
- **Ụzọ ngosi abụọ**: Widget akụkụ / Akpụkpọ elu, gbanwee site na pịa otu ugboro
- **Ngosi enwere ike ịhazi**: Họrọ ihe ị ga-egosi n'efu, gbanwee n'etiti pasent/uru na n'ezie
- **Na-anọ n'elu kọlezie**: Na-anọ n'elu windoo niile, na-arụ ọrụ na egwuregwu ihuenyo zuru ezu
- **Nha onwe ya**: Obosara windoo na-edozi onwe ya iji dabara ọdịnaya
- **Ụdị enwere ike ịgbanwe**: Ọhụụ windoo, n'azụ nke ọhụụ, nha fonti niile enwere ike ịgbanwe
- **Nkwado asụsụ dị iche iche**: Asụsụ 20+, na-achọpụta asụsụ sistemu na-akpaghị aka
- **Mmụta dị mfe**: Pịa ekpe iji bugharịa, pịa nri maka menu ntọala ngwa ngwa, na-edozi onwe ya na njedebe ihuenyo
- **Ọnọdụ nhọrọ enwere ike ịhazi**: 200ms - 3000ms enwere ike ịhazi
- **Malite onwe ya**: Malite na-akpaghị aka mgbe ịbanye na Windows
- **Obere ojiji akụrụngwa**: Obere mmetụ na ndabere

## Malite Ngwa Ọsịsọ

### Ji Ya Tụ̀ Mpụ̀

Budata `ChipPeek.exe` wee pịa okpukpu abụọ iji gbanye (ọ ga-arịọ ikike onye nchịkwa maka ịgụ ọnụ CPU na frekvensi ziri ezi).

### Gbaa Site na Isi Code

```bash
# Detuo ebe nchekwa
git clone https://github.com/Glimmer114514/ChipPeek.git
cd ChipPeek

# Wụnye ihe ndabere
pip install -r requirements.txt

# Gbaa
python main.py
```

## Ihe Ndị Dị Mkpa nke Sistemu

- Windows 10 / Windows 11
- Ikike onye nchịkwa (a na-atụ aro), ma ọ bụghị ya, ọnụ CPU na frekvensi ziri ezi nwere ike ịdị adị
- .NET Framework 4.7.2 ma ọ bụ karịa (LibreHardwareMonitor chọrọ ya)

## Bido Dị ka EXE

```bash
pip install pyinstaller
pyinstaller --noconfirm --onefile --windowed --name "ChipPeek" --icon "app.ico" --manifest "admin.manifest" --add-data "libs;libs" --add-data "app.ico;." --hidden-import "clr" --hidden-import "pynvml" --hidden-import "win32gui" --hidden-import "win32con" main.py
```

Mgbe ịchara ibido, faịlụ EXE dị na `dist/ChipPeek.exe`.

## Ojiji

### Mmụta Ndị Dị Mfe

| Mmụta | Nkọwa |
|------|------|
| Pịa ekpe | Bugharịa ebe windoo |
| Pịa nri | Mepee menu (gbanwee ọnọdụ / ntọala / pụọ) |
| Dozie onwe ya | Na-edozi onwe ya n'ime 20px nke njedebe ihuenyo |

### Ọnọdụ Ngosi

- **Widget akụkụ**: Emepụtara ihe ngosi niile n'ahịrị dị elu, enwere ike itinye ya n'akụkụ ọ bụla nke ihuenyo
- **Akpụkpọ elu**: Emepụtara ihe ngosi niile n'ahịrị dị larịị, n'etiti n'elu ihuenyo na ndabara

### Ntọala

- **Ọnọdụ ngosi**: Widget akụkụ / Akpụkpọ elu
- **Ebe widget**: Ala nke nri / Ala nke ekpe / Elu nke nri / Elu nke ekpe
- **Ọhụụ windoo**: 30% - 100% ọhụụ windoo n'ozuzu
- **N'azụ nke ọhụụ**: 0% - 100% n'azụ na-ahụ anya (ederede na-anọgide n'ụzọ doro anya)
- **Oge nhọrọ**: 200ms - 3000ms ọnụ ọgụgụ mmelite data
- **Nha fonti**: 8 - 20 isi fonti
- **Asụsụ**: Asụsụ 20+, na-achọpụta asụsụ sistemu na-akpaghị aka
- **Ihe ngosi**: Gbanwee ihe ọ bụla n'onwe ya
- **Ọdịdị ngosi**: VRAM / Echiche nwere ike ịgbanwe n'etiti pasent ma ọ bụ uru na n'ezie
- **Malite onwe ya**: Gbaa na-akpaghị aka mgbe ịbanye na Windows

## Nkà na ụzụ eji eme

- **GUI**：Tkinter
- **Data igwe**：LibreHardwareMonitorLib (site na pythonnet), psutil, pynvml (nhọrọ ndabere NVIDIA GPU)
- **Njikọta sistemu**：pywin32 (windoo dị elu, ndebata mmalite onwe ya)
- **Ibu pokétị**：PyInstaller
