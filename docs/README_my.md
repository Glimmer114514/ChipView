# ChipPeek

Windows အတွက် ပေါ့ပေါ့ဆ သော လွင့်ပျံသော ဟာ့ဒ်ဝဲ စောင့်ကြည့်မှု ဗီဂျက်။ CPU/GPU ကြိမ်နှုန်း၊ အပူချိန်၊ VRAM နှင့် မှတ်ဉာဏ်အသုံးပြုမှုကို အချိန်နှင့်တပြေးညီ စောင့်ကြည့်ပြီး ဖြည့်စွက်စခရင်အက်ပ်များတွင်ပင် အမြဲထိပ်တွင် ရှိနေသည်။

## အင်္ဂါရပ်များ

- **အချိန်နှင့်တပြေးညီ စောင့်ကြည့်မှု**: CPU ကြိမ်နှုန်း၊ CPU အပူချိန်၊ GPU ကြိမ်နှုန်း၊ GPU အပူချိန်၊ VRAM အသုံးပြုမှု၊ မှတ်ဉာဏ်အသုံးပြုမှု
- **ဖော်ပြမှုပုံစံ နှစ်မျိုး**: ထောင့်ဗီဂျက် / ထိပ်ဘား၊ ကလစ်တစ်ချက်နှင့်ပြောင်းလဲ
- **စိတ်ကြိုက် ဖော်ပြမှု**: မည်သည့် မက်ထရစ်များကို ပြသမည်ကို လွတ်လပ်စွာ ရွေးချယ်ပြီး ရာခိုင်နှုန်း/အစစ်တန်ဖိုးများ ကြားဖြင့် ပြောင်းလဲ
- **အမြဲထိပ်တွင်**: ဝင်းဒိုးအားလုံး၏ ထိပ်တွင် ရှိနေပြီး ဖြည့်စွက်စခရင်ဂိမ်းများတွင် အလုပ်လုပ်သည်
- **အလိုအလျောက် အရွယ်အစား**: အကြောင်းအရာပေါ်မူတည်၍ ဝင်းဒိုးအကျယ် အလိုအလျောက် ချိန်ညှိသည်
- **ချိန်ညှိနိုင်သော စတိုင်**: ဝင်းဒိုးပွင့်လင်းမှု၊ နောက်ခံပွင့်လင်းမှု၊ ဖောင့်အရွယ်အစား - အားလုံး ချိန်ညှိနိုင်သည်
- **ဘာသာစကားပေါင်းစုံ ပံ့ပိုးမှု**: 20+ ဘာသာစကား၊ စနစ်ဘာသာစကားကို အလိုအလျောက် ရှာဖွေတွေ့ရှိသည်
- **လွယ်ကူသော လုပ်ဆောင်ချက်**: ဘယ်ဖက်ကလစ်ဖြင့် ဆွဲချပြီး ရွှေ့ပါ၊ ညာဖက်ကလစ်မီနူးတွင် လျင်မြန်သော ဆက်တင်များအတွက်၊ ဖန်သားအစွန်းများတွင် အလိုအလျောက် ကပ်ခ်
- **နမူနာယူခြင်း ကြိမ်နှုန်း ချိန်ညှိခြင်း**: 200ms - 3000ms ချိန်ညှိနိုင်သည်
- **အလိုအလျောက် စတင်ခြင်း**: Windows သို့ ဝင်ရောက်သည့်အခါ အလိုအလျောက် စတင်သည်
- **အရင်းအမြစ် အသုံးပြုမှု နည်းပါးခြင်း**: နောက်ခံတွင် အနည်းဆုံး နေရာ

## လျင်မြန်စွာ စတင်ခြင်း

### တိုက်ရိုက်အသုံးပြုခြင်း

`ChipPeek.exe` ကို ဒေါင်းလုဒ်လုပ်ပြီး run ဖို့ နှစ်ချက်ကလစ်လုပ်ပါ (CPU အပူချိန်နှင့် တိကျသောကြိမ်နှုန်းကို ဖတ်ရှုရန် အလိုအလျောက် အက်ဒမင်ခွင့်ပြုချက်တောင်းဆိုမည်)။

### ရင်းမြစ်ကုဒ်မှ run ခြင်း

```bash
# စတိုးချုပ်ကို ကလုန်လုပ်ပါ
git clone https://github.com/Glimmer114514/ChipPeek.git
cd ChipPeek

# မှီခိုမှုများ တပ်ဆင်ပါ
pip install -r requirements.txt

# run ခြင်း
python main.py
```

## စနစ် လိုအပ်ချက်များ

- Windows 10 / Windows 11
- အက်ဒမင်ခွင့်ပြုချက် (အကြံပြုထားသည်)၊ မဟုတ်လျှင် CPU အပူချိန်နှင့် တိကျသောကြိမ်နှုန်း မရနိုင်ပါ
- .NET Framework 4.7.2 သို့မဟုတ် အထက် (LibreHardwareMonitor အတွက် လိုအပ်သည်)

## EXE အဖြစ် တည်ဆောက်ခြင်း

```bash
pip install pyinstaller
pyinstaller --noconfirm --onefile --windowed --name "ChipPeek" --icon "app.ico" --manifest "admin.manifest" --add-data "libs;libs" --add-data "app.ico;." --hidden-import "clr" --hidden-import "pynvml" --hidden-import "win32gui" --hidden-import "win32con" main.py
```

တည်ဆောက်ပြီးနောက်၊ EXE ဖိုင်သည် `dist/ChipPeek.exe` တွင် ရှိသည်။

## အသုံးပြုနည်း

### အခြေခံ လုပ်ဆောင်ချက်များ

| လုပ်ဆောင်ချက် | ဖော်ပြချက် |
|------|------|
| ဘယ်ဖက်ကလစ်ဖြင့် ဆွဲခြင်း | ဝင်းဒိုးတည်နေရာ ရွှေ့ခြင်း |
| ညာဖက်ကလစ် | မီနူးဖွင့်ခြင်း (မုဒ်ပြောင်းလဲခြင်း / ဆက်တင်များ / ထွက်ခြင်း) |
| အလိုအလျောက် ကပ်ခ် | ဖန်သားအစွန်းများမှ 20px အတွင်းသို့ အလိုအလျောက် ကပ်သည် |

### ဖော်ပြမှုပုံစံ များ

- **ထောင့်ဗီဂျက်**: မက်ထရစ်အားလုံးကို ဒေါင်လိုက်စီ စီစဉ်ထားပြီး ဖန်သား၏ မည်သည့်ထောင့်တွင်မှ ထားရှိနိုင်သည်
- **ထိပ်ဘား**: မက်ထရစ်အားလုံးကို အလျားလိုက်စီ စီစဉ်ထားပြီး ပုံသေအားဖြင့် ဖန်သား၏ထိပ်ဘက် အလယ်တွင် ရှိသည်

### ဆက်တင်များ

- **ဖော်ပြမှုပုံစံ**: ထောင့်ဗီဂျက် / ထိပ်ဘား
- **ဗီဂျက်တည်နေရာ**: အောက်ညာ / အောက်ဘယ် / ထိပ်ညာ / ထိပ်ဘယ်
- **ဝင်းဒိုးပွင့်လင်းမှု**: 30% - 100% စုစုပေါင်း ဝင်းဒိုးပွင့်လင်းမှု
- **နောက်ခံပွင့်လင်းမှု**: 0% - 100% နောက်ခံပွင့်လင်းမှု (စာသားသည် ရှင်းလင်းစွာ ရှိနေသည်)
- **နမူနာယူခြင်း ကြားကာလ**: 200ms - 3000ms ဒေတာ သက်တမ်းတိုးနှုန်း
- **ဖောင့်အရွယ်အစား**: 8 - 20 အမှတ်များ
- **ဘာသာစကား**: 20+ ဘာသာစကား၊ စနစ်ဘာသာစကားကို အလိုအလျောက် ရှာဖွေတွေ့ရှိသည်
- **ဖော်ပြမှု မက်ထရစ်များ**: မက်ထရစ်တစ်ခုစီကို လွတ်လပ်စွာ ဖွင့်/ပိတ်နိုင်သည်
- **ဖော်ပြမှု ပုံစံ**: VRAM / မှတ်ဉာဏ် ရာခိုင်နှုန်း သို့မဟုတ် အစစ်တန်ဖိုးများ ကြားဖြင့် ပြောင်းလဲနိုင်သည်
- **အလိုအလျောက် စတင်ခြင်း**: Windows သို့ ဝင်ရောက်သည့်အခါ အလိုအလျောက် run ခြင်း

## နည်းပညာ အစုအဝေး

- **GUI**：Tkinter
- **ဟာ့ဒ်ဝဲ ဒေတာ**：LibreHardwareMonitorLib (pythonnet မှတဆင့်), psutil, pynvml (NVIDIA GPU အတွက် အရန်)
- **စနစ် ပေါင်းစပ်မှု**：pywin32 (ဝင်းဒိုး အမြဲထိပ်တွင်၊ အလိုအလျောက်စတင်ခြင်း စာရင်း)
- **ထုပ်ပိုးခြင်း**：PyInstaller
