## លក្ខណៈពិសេស

- **តាមដានពេលជាក់ស្តែង**: ប្រេកង់ CPU សីតុណ្ហភាព CPU ប្រេកង់ GPU សីតុណ្ហភាព GPU ការប្រើប្រាស់ VRAM ការប្រើប្រាស់មេម៉ូរី
- **របៀបបង្ហាញពីរ**: វីជេតមុំ / របារខាងលើ ប្តូរដោយការចុចតែមួយ
- **ការបង្ហាញផ្ទាល់ខ្លួន**: ជ្រើសរើសសេរីថាតើមែទ្រីកណាត្រូវបង្ហាញ ប្តូររវាងភាគរយ/តម្លៃពិត
- **នៅលើកំពូលជានិច្ច**: នៅលើកំពូលនៃបង្អួចទាំងអស់ ធ្វើការនៅក្នុងហ្គេមអេក្រង់ពេញ
- **ទំហំស្វ័យប្រវត្តិ**: ប្រវែងបង្អួចត្រូវបានកែសំរួលដោយស្វ័យប្រវត្តិយោងទៅលើមាតិកា
- **រចនាបថដែលអាចកែសំរួលបាន**: ភាពចំហរបស់បង្អួច ភាពចំហរបស់ផ្ទៃខាងក្រោយ ទំហំហ្វុង - ទាំងអស់អាចកែសំរួលបាន
- **ការគាំទ្រពហុភាសា**: 20+ ភាសា រកឃើញភាសាប្រព័ន្ធដោយស្វ័យប្រវត្តិ
- **ប្រតិបត្តិការងាយស្រួល**: អូសដោយការចុចឆ្វេងដើម្បីផ្លាស់ទី ម៉ឺនុយចុចស្តាំសម្រាប់ការកំណត់រហ័ស ខ្ទៀងផ្ទាត់ស្វ័យប្រវត្តិនៅគែមអេក្រង់
- **ការកែសំរួលប្រេកង់គំរូ**: 200ms - 3000ms អាចកែសំរួលបាន
- **ចាប់ផ្តើមស្វ័យប្រវត្តិ**: ចាប់ផ្តើមដោយស្វ័យប្រវត្តិនៅពេលចូល Windows
- **ការប្រើប្រាស់ធនធានទាប**: កាន់កាប់កន្លែងអប្បបរមានៅផ្ទៃខាងក្រោយ

## ការចាប់ផ្តើមរហ័ស

### ការប្រើប្រាស់ផ្ទាល់

ទាញយក `ChipPeek.exe` ហើយចុចពីរដងដើម្បីដំណើរការ (វានឹងស្នើសុំសិទ្ធិអ្នកគ្រប់គ្រងដោយស្វ័យប្រវត្តិដើម្បីអានសីតុណ្ហភាព CPU និងប្រេកង់ជាក់លាក់)។

### ដំណើរការពីកូដដើម

```bash
# ក្លូនឃ្លាំង
git clone https://github.com/Glimmer114514/ChipPeek.git
cd ChipPeek

# ដំឡើងអាស្រ័យ
pip install -r requirements.txt

# ដំណើរការ
python main.py
```

## តម្រូវការប្រព័ន្ធ

- Windows 10 / Windows 11
- សិទ្ធិអ្នកគ្រប់គ្រង (បានណែនាំ) បើមិនដូច្នេះទេ សីតុណ្ហភាព CPU និងប្រេកង់ជាក់លាក់អាចមិនមាន
- .NET Framework 4.7.2 ឬខ្ពស់ជាង (ត្រូវការសម្រាប់ LibreHardwareMonitor)

## បង្កើតជា EXE

```bash
pip install pyinstaller
pyinstaller --noconfirm --onefile --windowed --name "ChipPeek" --icon "app.ico" --manifest "admin.manifest" --add-data "libs;libs" --add-data "app.ico;." --hidden-import "clr" --hidden-import "pynvml" --hidden-import "win32gui" --hidden-import "win32con" main.py
```

បន្ទាប់ពីបង្កើតរួច ឯកសារ EXE ស្ថិតនៅ `dist/ChipPeek.exe`។

- **វីជេតមុំ**: មែទ្រីកទាំងអស់ត្រូវបានរៀបជាបញ្ឈរ អាចដាក់នៅជ្រុងណាមួយនៃអេក្រង់បាន
- **របារខាងលើ**: មែទ្រីកទាំងអស់ត្រូវបានរៀបជាផ្ដេក លំនាំដើមស្ថិតនៅកណ្តាលផ្នែកខាងលើនៃអេក្រង់

### ការកំណត់

- **របៀបបង្ហាញ**: វីជេតមុំ / របារខាងលើ
- **ទីតាំងវីជេត**: ខាងស្តាំខាងក្រោម / ខាងឆ្វេងខាងក្រោម / ខាងស្តាំខាងលើ / ខាងឆ្វេងខាងលើ
- **ភាពចំហរបស់បង្អួច**: 30% - 100% ភាពចំហបង្អួចសរុប
- **ភាពចំហរបស់ផ្ទៃខាងក្រោយ**: 0% - 100% ភាពចំហផ្ទៃខាងក្រោយ (អត្ថបទនៅជាប់ល្អ)
- **ចន្លោះគំរូ**: 200ms - 3000ms អត្រាធ្វើឱ្យទាន់សម័យទិន្នន័យ
- **ទំហំហ្វុង**: 8 - 20 ចំនុច
- **ភាសា**: 20+ ភាសា រកឃើញភាសាប្រព័ន្ធដោយស្វ័យប្រវត្តិ
- **មែទ្រីកបង្ហាញ**: បើក/បិទមែទ្រីកនីមួយៗដោយឯករាជ្យ
- **ទម្រង់បង្ហាញ**: VRAM / មេម៉ូរីអាចប្តូររវាងភាគរយ ឬតម្លៃពិតបាន
- **ចាប់ផ្តើមស្វ័យប្រវត្តិ**: ដំណើរការដោយស្វ័យប្រវត្តិនៅពេលចូល Windows

## ជុំវិញបច្ចេកវិទ្យា

- **GUI**：Tkinter
- **ទិន្នន័យផ្នែករឹង**：LibreHardwareMonitorLib (តាមរយៈ pythonnet), psutil, pynvml (ការបម្រុងទុកសម្រាប់ NVIDIA GPU)
- **ការធ្វើសមាហរណកម្មប្រព័ន្ធ**：pywin32 (បង្អួចនៅលើកំពូលជានិច្ច ការចុះបញ្ជីចាប់ផ្តើមស្វ័យប្រវត្តិ)
- **ការវេចខ្ចប់**：PyInstaller

## រចនាសម្ព័ន្ធគម្រោង

```
ChipPeek/
├── main.py                  # ចំណុចប្រទាក់ចូល
├── monitor_window.py        # បង្អួច UI និងតក្កវិជ្ជាអន្តរកម្ម
├── hardware_monitor.py      # ការប្រមូលទិន្នន័យផ្នែករឹង
├── config.py                # ការគ្រប់គ្រងការកំណត់រចនាសម្ព័ន្ធ
├── i18n.py                  # អន្តរជាតិ
├── generate_icon.py         # ស្គ្រីបបង្កើតរូបតំណាង
├── admin.manifest           # UAC មនិផេសថ្លៃសិទ្ធិអ្នកគ្រប់គ្រង
├── app.ico                  # រូបតំណាងកម្មវិធី
├── app.png                  # ការមើលជាមុនរូបតំណាង
├── requirements.txt         # អាស្រ័យ Python
├── docs/                    # ឯកសារពហុភាសា
│   └── README_*.md
├── i18n/                    # ឯកសារបកប្រែ
│   └── *.json
├── libs/
│   └── lhm/
│       └── lib/net472/      # LibreHardwareMonitorLib DLL
└── dist/
    └── ChipPeek.exe         # ឯកសារប្រតិបត្តិការដែលបានបង្កើត
```
