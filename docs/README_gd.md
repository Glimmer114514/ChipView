## Gleusan

- **Monataireadh fìor-ùine**: Tricead CPU, teòthachd CPU, tricead GPU, teòthachd GPU, cleachdadh VRAM, cleachdadh cuimhne
- **Dà mhodh taisbeanaidh**: Bidean ceàrnrainn / Bàr-bàrr, atharrachadh le aon bhriog
- **Taisbeanadh gnàthaichte**: Tagh saor an làraichean ri sealltainn, toglach eadar ceudad/luachan iar-fìor
- **An-còmhnaidh air a' bharr**: Còmhdaich air uile na h-uinneagan, obraichidh ann an geamannan làn-sgrìn
- **Meud fhèin-gluasadach**: Leud na h-uinneige a' atharrachadh fhèin a rèir susbaint
- **Stoidhle rèiteachail**: Trìd-shoilleireachd uinneige, trìd-shoilleireachd cùlaibh, meud a' chruth-chlò uile rèiteachail
- **Taic ioma-chànanach**: 20+ cànan, mothachadh fèin-air systèam-chànan
- **Gnìomh chomhfhurtail**: Slaod chlì airson gluasad, clàr-taice briog deas airson roghainnean luatha, glèidh fhèin ri oirean sgrìn
- **Eadaramh samplachaidh rèiteachail**: 200ms - 3000ms rèiteachail
- **Fèin-tòiseachadh**: Tòisich fhèin nuair a chlàraichear a-steach gu Windows
- **Cleachdadh goireas ìseal**: Lùgh beag anns a' chùlaibh

## Tòiseachadh luath

### Cleachdadh dìreach

Luchdaich a-nuas `ChipPeek.exe` agus dèan briogadh dùbailte airson ruith (iarridh e fèin-umhail ceartan rianaire airson teòthachd CPU agus tricead pongail).

### Ruith on chòd bunait

```bash
# Clone the repository
git clone https://github.com/Glimmer114514/ChipPeek.git
cd ChipPeek

# Install dependencies
pip install -r requirements.txt

# Run
python main.py
```

## Riatanasan an t-siostaim

- Windows 10 / Windows 11
- Ceartan rianaire (air a mholadh), air neo dh'fhaodadh teòthachd CPU agus tricead pongail a bhith neo-ruigsinneach
- .NET Framework 4.7.2 no nas àirde (ga iarradh le LibreHardwareMonitor)

## Tog mar EXE

```bash
pip install pyinstaller
pyinstaller --noconfirm --onefile --windowed --name "ChipPeek" --icon "app.ico" --manifest "admin.manifest" --add-data "libs;libs" --add-data "app.ico;." --hidden-import "clr" --hidden-import "pynvml" --hidden-import "win32gui" --hidden-import "win32con" main.py
```

Às dèidh a thogail, tha am faidhle EXE suidhichte aig `dist/ChipPeek.exe`.

## Cleachdadh

### Oibrichean bunaiteach

| Gnìomh | Tuairisgeul |
|------|------|
| Slaod chlì | Gluais ionad na h-uinneige |
| Briog deas | Fosgail clàr-taice (atharraich am modh / roghainnean / fàg) |
| Glèidh fhèin | Glèidh fhèin taobh a-staigh 20px de oirean an sgrìn |

### Modhan taisbeanaidh

- **Bidean ceàrnrainn**: A h-uile nì rèitichte gu h-inghearach, faodar a chur ann an ceàrn sam bith den sgrìn
- **Bàr-bàrr**: A h-uile nì rèitichte gu còmhnard, meadhanach aig mullach na sgrìn a ghnàth

### Roghainnean

- **Modh taisbeanaidh**: Bidean ceàrnrainn / Bàr-bàrr
- **Ionad a' bhidein**: Bun deas / Bun clì / Bàrr deas / Bàrr clì
- **Trìd-shoilleireachd uinneige**: 30% - 100% trìd-shoilleireachd iomlan na h-uinneige
- **Trìd-shoilleireachd cùlaibh**: 0% - 100% cùlaibh trìd-shoilleir (tha teacsa fhathast soilleir)
- **Eadaramh samplachaidh**: 200ms - 3000ms reat ùrachaidh dàta
- **Meud a' chruth-chlò**: 8 - 20 peant cruth-chlò
- **Cànan**: 20+ cànan, mothachadh fèin-air systèam-chànan
- **Nithean taisbeanaidh**: Toglaich gach nì leis fhèin
- **Fòrmat taisbeanaidh**: Faodar VRAM / Cuimhne a thoglach eadar ceudad no luachan iar-fìor
- **Fèin-tòiseachadh**: Ruich fhèin nuair a chlàraichear a-steach gu Windows

## Stac teicneòlais

- **GUI**：Tkinter
- **Dàta bathar-cruaidh**：LibreHardwareMonitorLib (tro pythonnet), psutil, pynvml (GPU NVIDIA fallback)
- **Amalachadh an t-siostaim**：pywin32 (uinneag air a' bharr, clàr fèin-tòiseachaidh)
- **Pacadadh**：PyInstaller

├── generate_icon.py         # Sgriobt gineadh ìomhaighe

├── app.ico                  # Ìomhaigh a' phrògraim
├── app.png                  # Ro-shealladh an ìomhaigh

├── docs/                    # Docamaidean ioma-chànanach
